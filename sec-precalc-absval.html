    Subsection Important Definitions and Theorems

    Definition 0.2.1. Absolute Value.

    For any real number \(A\text{,}\) the absolute value of \(A\text{,}\) written \(|A|\text{,}\) is defined by
    ğŸ”—
    \begin{equation*} |A| = \begin{cases} A, \amp \text{if } A \geq 0, \\\\ -A, \amp \text{if } A \le 0. \end{cases} \end{equation*}
    Geometrically, \(|A|\) represents the distance from \(A\) to \(0\) on the number line.

    Theorem 0.2.2.

    If \(|A|=k\) then \(A=k\) or \(A=-k\text{.}\)
