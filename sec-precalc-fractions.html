    Example 0.1.5.

    Simplify the expression as much as possible: \(\dfrac{2x^2 - 11x + 5}{4x^2 - 1}\)
    ğŸ”—
    Answer.
    \begin{equation*} \dfrac{x-5}{2x+1} \end{equation*}
    ğŸ”—
    ğŸ”—
    Solution.
    1. Factor the numerator \(2x^2-11x+5\) using the \(ac\)-method.
      ğŸ”—
      • \(a=2\text{,}\) \(c=5\) so \(ac=10\text{.}\) We need two numbers that multiply to \(10\) and add to \(-11\text{:}\) \(-10\) and \(-1\text{.}\)
        ğŸ”—
        ğŸ”—
      • Split the middle term and factor by grouping:
        ğŸ”—
        \(\begin{align*} 2x^2 - 11x + 5 \amp= 2x^2 - 10x - x + 5 \\ \amp= (2x^2 - 10x) - (x - 5) \\ \amp= 2x(x-5) - 1(x-5) \\ \amp= (2x-1)(x-5). \end{align*}\)
        ğŸ”—
        ğŸ”—
      ğŸ”—
    2. Factor the denominator \(4x^2-1\) as a difference of squares:
      ğŸ”—
      \(4x^2-1 = (2x)^2-1^2 = (2x-1)(2x+1).\)
      ğŸ”—
      ğŸ”—

      ğŸ”—

      ğŸ”—
      ğŸ”—
    4. Conclusion. The simplified form is \(\displaystyle \frac{x-5}{2x+1}\)

    Example 0.1.6.

    Simplify the expression as much as possible: \(\dfrac{x^2 - 2x - 8}{x^2 + 2x - 8}\)
    ğŸ”—
    Answer.
    \begin{equation*} \dfrac{(x+2)(x-4)}{(x-2)(x+4)} \end{equation*}
    ğŸ”—
    ğŸ”—
    Solution.
    1. Factor the numerator \(x^2 - 2x - 8\text{.}\)
      ğŸ”—
      Look for two numbers that multiply to \(-8\) (the constant term) and add to \(-2\) (the coefficient of \(x\)). These are \(-4\) and \(+2\text{.}\) So:
      ğŸ”—
      \begin{equation*} x^2 - 2x - 8 = (x - 4)(x + 2). \end{equation*}
      ğŸ”—
    2. Factor the denominator \(x^2 + 2x - 8\text{.}\)
      ğŸ”—
      Look for two numbers that multiply to \(-8\) and add to \(+2\text{.}\) These are \(+4\) and \(-2\text{.}\) So:
      ğŸ”—
      \begin{equation*} x^2 + 2x - 8 = (x + 4)(x - 2). \end{equation*}
      ğŸ”—
    3. Write the fraction with factors:
      ğŸ”—
      \begin{equation*} \frac{x^2 - 2x - 8}{x^2 + 2x - 8} = \frac{(x - 4)(x + 2)}{(x + 4)(x - 2)}. \end{equation*}
      ğŸ”—
    4. Check for common factors. None of the factors cancel, so this is as simplified as it gets.
      ğŸ”—
      ğŸ”—
    5. Conclusion. The simplified form is \(\dfrac{(x+2)(x-4)}{(x-2)(x+4)}\text{.}\)

    Example 0.1.7.

    Simplify the expression as much as possible: \(\dfrac{\tfrac{1}{\sqrt{x}} - \tfrac{1}{2}}{x-4}\)
    ğŸ”—
    Answer.
    \begin{equation*} \frac{-1}{4\sqrt{x} + 2x}. \end{equation*}
    ğŸ”—
    ğŸ”—
    Solution.
    1. Combine the terms in the numerator.
      ğŸ”—
      To combine \(\tfrac{1}{\sqrt{x}}\) and \(\tfrac{1}{2}\text{,}\) use a common denominator of \(2\sqrt{x}\text{:}\)
      ğŸ”—
      \begin{equation*} \frac{1}{\sqrt{x}} - \frac{1}{2} = \frac{2}{2\sqrt{x}} - \frac{\sqrt{x}}{2\sqrt{x}} = \frac{2 - \sqrt{x}}{2\sqrt{x}}. \end{equation*}
      ğŸ”—
    2. Rewrite the entire fraction.
      ğŸ”—
      \begin{equation*} \frac{\tfrac{1}{\sqrt{x}} - \tfrac{1}{2}}{x-4} = \frac{\tfrac{2 - \sqrt{x}}{2\sqrt{x}}}{x-4} = \frac{2 - \sqrt{x}}{2\sqrt{x}(x-4)}. \end{equation*}
      ğŸ”—
    3. Simplify the numerator using the conjugate.
      ğŸ”—
      Multiply top and bottom by the conjugate \(2 + \sqrt{x}\) to eliminate the radical in \(2 - \sqrt{x}\text{:}\)
      ğŸ”—
      \begin{equation*} \frac{2 - \sqrt{x}}{2\sqrt{x}(x-4)} \cdot \frac{2 + \sqrt{x}}{2 + \sqrt{x}} = \frac{(2 - \sqrt{x})(2 + \sqrt{x})}{2\sqrt{x}(x-4)(2+\sqrt{x})}. \end{equation*}
      ğŸ”—
    4. Simplify the numerator product.
      ğŸ”—
      \((2 - \sqrt{x})(2 + \sqrt{x}) = 4 - x.\)
      ğŸ”—
      \begin{equation*} \frac{4 - x}{2\sqrt{x}(x-4)(2+\sqrt{x})}. \end{equation*}
      ğŸ”—
    5. Factor out \(-1\) from \(4 - x\text{.}\)
      ğŸ”—
      \begin{equation*} 4 - x = -(x - 4), \end{equation*}
      so the fraction becomes:
      ğŸ”—
      \begin{equation*} \frac{-(x-4)}{2\sqrt{x}(x-4)(2+\sqrt{x})}. \end{equation*}
      ğŸ”—
    6. Cancel the common factor \((x-4)\) (with the restriction \(x \neq 4\)):
      ğŸ”—
      \begin{equation*} \frac{-1}{2\sqrt{x}(2+\sqrt{x})}. \end{equation*}
      ğŸ”—
    7. Simplify the denominator.
      ğŸ”—
      Expand \(2\sqrt{x}(2+\sqrt{x})\text{:}\)
      ğŸ”—
      \begin{equation*} 2\sqrt{x}(2+\sqrt{x}) = 4\sqrt{x} + 2x. \end{equation*}
      So the expression becomes:
      ğŸ”—
      \begin{equation*} \frac{-1}{4\sqrt{x} + 2x}. \end{equation*}
      ğŸ”—
    8. Conclusion. A simplified form is \(\frac{-1}{4\sqrt{x} + 2x}.\)

    Example 0.1.11.

    Simplify the expression as much as possible: \(\dfrac{8x}{x^2 - 6x + 5} - \dfrac{x-7}{x^2 + x - 2}\)
    ğŸ”—
    Answer.
    \begin{equation*} \dfrac{7(x+5)}{(x+2)(x-5)} \end{equation*}
    ğŸ”—
    ğŸ”—
    Solution.
    1. Factor the denominators.
      ğŸ”—
      \begin{equation*} x^2 - 6x + 5 = (x-1)(x-5), \end{equation*}
      \begin{equation*} x^2 + x - 2 = (x+2)(x-1). \end{equation*}
      ğŸ”—
    2. Use the LCD \((x-1)(x-5)(x+2)\) and rewrite each fraction.
      ğŸ”—
      \begin{equation*} \frac{8x}{(x-1)(x-5)} = \frac{8x(x+2)}{(x-1)(x-5)(x+2)}, \end{equation*}
      \begin{equation*} \frac{x-7}{(x+2)(x-1)} = \frac{(x-7)(x-5)}{(x-1)(x-5)(x+2)}. \end{equation*}
      Therefore
      ğŸ”—
      \begin{equation*} \frac{8x}{x^2 - 6x + 5} - \frac{x-7}{x^2 + x - 2} = \frac{8x(x+2) - (x-7)(x-5)}{(x-1)(x-5)(x+2)}. \end{equation*}
      ğŸ”—
    3. Expand and simplify the numerator.
      ğŸ”—
      \(\begin{align*} 8x(x+2) - (x-7)(x-5) \amp= (8x^2 + 16x) - (x^2 - 12x + 35) \\ \amp= 8x^2 + 16x - x^2 + 12x - 35 \\ \amp= 7x^2 + 28x - 35 \\ \amp= 7(x^2 + 4x - 5) \\ \amp= 7(x+5)(x-1). \end{align*}\)
      ğŸ”—
    4. Cancel common factors and conclude.
      ğŸ”—
      \begin{equation*} \frac{7(x+5)(x-1)}{(x-1)(x-5)(x+2)} = \frac{7(x+5)}{(x-5)(x+2)} = \frac{7(x+5)}{(x+2)(x-5)}. \end{equation*}
