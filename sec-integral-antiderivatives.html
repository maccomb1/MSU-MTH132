    Section 4.1 Antiderivatives

    Objectives

    • Interpret an antiderivative as “reversing differentiation” and connect it to rates of change (velocity, acceleration).
      🔗
    • Find general antiderivatives using basic antidifferentiation rules and linearity.
      🔗
    • Solve simple initial value problems by using a given function value to determine the constant of integration.
      🔗
    • Use simple substitution patterns (reverse Chain Rule) when a constant multiple is missing.
      🔗
    • Recognize when an integrand must be rewritten (for example, by algebra or a trig identity) before antidifferentiating.
      🔗
    🔗

    Subsection Reversing Differentiation

    In many applications we care about some unknown function \(F(x)\text{,}\) but what we can measure or model is its rate of change, the derivative \(F'(x)=f(x)\text{.}\) Finding \(F\) from \(f\) is the process of antidifferentiation. We call \(F\) an antiderivative (or primitive) of \(f\text{.}\) A helpful memory trick is “big \(F\text{,}\) little \(f\text{.}\)”
    🔗

    Example 4.1.1.

    Suppose \(F'(t)=t^2-1\text{.}\) Find a general formula for \(F(t)\text{.}\)
    🔗
    Solution.
    Since \((\tfrac13 t^3)'=t^2\) and \((-t)'=-1\text{,}\) one antiderivative is \(\tfrac13 t^3-t\text{.}\) Any constant disappears when differentiating, so the general antiderivative is
    🔗
    \begin{equation*} F(t)=\tfrac13 t^3-t+C. \end{equation*}
    🔗
    🔗
    Different choices of \(C\) shift the graph up or down but do not change slopes. If we also know a single value like \(F(0)=5\text{,}\) then we can determine \(C\) and get a unique function. Problems like this are called initial value problems.
    🔗

    Example 4.1.2.

    Find the unique function \(F\) satisfying \(F'(t)=t^2+2\) and \(F(0)=5\text{.}\)
    🔗
    Solution.
    An antiderivative of \(t^2+2\) is \(\tfrac13 t^3+2t\text{,}\) so \(F(t)=\tfrac13 t^3+2t+C\text{.}\) Plug in \(t=0\text{:}\) \(F(0)=C=5\text{.}\)
    🔗
    \begin{equation*} F(t)=\tfrac13 t^3+2t+5. \end{equation*}
    🔗
    🔗
    🔗

    Subsection Definition and the Constant of Integration

    Definition 4.1.3. Antiderivative.

    A function \(F(x)\) is an antiderivative of \(f(x)\) on an interval if \(F'(x)=f(x)\) for all \(x\) in that interval.
    🔗
    🔗

    Theorem 4.1.4. Antiderivatives Differ by a Constant.

    Suppose \(F_0(x)\) is one antiderivative of \(f(x)\) on an interval. Then every antiderivative of \(f\) has the form \(F(x)=F_0(x)+C\) for some constant \(C\text{.}\)
    🔗
    🔗

    Proof.

    If \(F\) is any antiderivative of \(f\text{,}\) then \(F'(x)=f(x)=F_0'(x)\text{.}\) By the Uniqueness Theorem from the Mean Value Theorem section, two functions with the same derivative differ by a constant. Therefore \(F(x)=F_0(x)+C\text{.}\)
    🔗
    🔗

    Theorem 4.1.5. Uniqueness with One Function Value.

    If \(F_0\) is an antiderivative of \(f\) on an interval, then for any point \(c\) in the interval and any real number \(A\text{,}\) there is exactly one antiderivative \(F\) satisfying \(F'(x)=f(x)\) and \(F(c)=A\text{.}\)
    🔗
    🔗

    Proof.

    By the previous theorem, any antiderivative has the form \(F(x)=F_0(x)+C\text{.}\) Requiring \(F(c)=A\) gives \(F_0(c)+C=A\text{,}\) so \(C=A-F_0(c)\text{,}\) which is a single value.
    🔗
    🔗
    🔗

    Subsection Basic Antidifferentiation Rules

    Antidifferentiation starts by reversing the derivative rules you already know. The key is to build a library of basic patterns, then use linearity to break complicated expressions into simpler pieces.
    🔗

    Theorem 4.1.6. Linearity of Antiderivatives.

    If \(F'(x)=f(x)\) and \(G'(x)=g(x)\text{,}\) then for constants \(a,b\text{,}\) an antiderivative of \(af(x)+bg(x)\) is \(aF(x)+bG(x)\text{.}\)
    🔗
    🔗
    Here are several basic patterns (each includes the constant of integration).
    🔗
    \begin{equation*} \begin{aligned} \text{Power rule:}\quad &\text{if } n\neq -1,\ \ \int x^n\,dx \text{ corresponds to } \frac{1}{n+1}x^{n+1}+C.\\ \text{Special case:}\quad &\text{an antiderivative of } \frac{1}{x} \text{ is } \ln|x|+C.\\ \text{Trig:}\quad &\int \cos(x)\,dx \text{ corresponds to } \sin(x)+C,\qquad \int \sin(x)\,dx \text{ corresponds to } -\cos(x)+C.\\ &\int \sec^2(x)\,dx \text{ corresponds to } \tan(x)+C,\qquad \int \sec(x)\tan(x)\,dx \text{ corresponds to } \sec(x)+C. \end{aligned} \end{equation*}
    We are not formally defining integrals yet, so you can read the display above as a list of “reverse derivative” facts. The important idea is that the derivative table can be read backward, with a \(+C\) added.
    🔗

    Example 4.1.7.

    Find a general antiderivative of \(f(x)=7x^3-x\sqrt{x}+\dfrac{3}{x^2}-\dfrac{4}{\cos^2(x)}\text{.}\)
    🔗
    Solution.
    Rewrite using exponents and secant: \(x\sqrt{x}=x^{3/2}\text{,}\) \(\dfrac{3}{x^2}=3x^{-2}\text{,}\) and \(\dfrac{1}{\cos^2(x)}=\sec^2(x)\text{.}\) Then antidifferentiate term-by-term:
    🔗
    \begin{equation*} \begin{aligned} F(x) &= 7\cdot \frac{1}{4}x^4 \;-\; \frac{1}{\frac52}x^{5/2} \;+\; 3\cdot\frac{1}{-1}x^{-1} \;-\;4\tan(x) + C\\[0.3em] &= \frac{7}{4}x^4 \;-\; \frac{2}{5}x^{5/2} \;-\;\frac{3}{x}\;-\;4\tan(x)+C. \end{aligned} \end{equation*}
    A quick check is to differentiate \(F(x)\) and confirm you get back \(f(x)\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Subsection A Simple “Reverse Chain Rule” Pattern

    Sometimes a function looks like the derivative of a composition, except for a missing constant multiple. In these cases, you can often fix the issue by introducing the correct constant.
    🔗

    Example 4.1.8.

    Find a general antiderivative of \(f(x)=\cos(3x)\text{.}\)
    🔗
    Solution.
    Since \((\sin(3x))'=3\cos(3x)\text{,}\) we divide by 3:
    🔗
    \begin{equation*} F(x)=\tfrac13\sin(3x)+C. \end{equation*}
    🔗
    🔗

    Example 4.1.9.

    Find a general antiderivative of \(\displaystyle f(x)=\frac{x+4}{\sqrt{x}}\text{.}\)
    🔗
    Solution.
    Rewrite as a sum of powers: \(\displaystyle \frac{x+4}{\sqrt{x}}=\frac{x}{\sqrt{x}}+\frac{4}{\sqrt{x}}=x^{1/2}+4x^{-1/2}\text{.}\) Then antidifferentiate term-by-term:
    🔗
    \begin{equation*} \begin{aligned} F(x) &= \frac{1}{\frac32}x^{3/2}+4\cdot\frac{1}{\frac12}x^{1/2}+C\\ &= \tfrac23 x\sqrt{x}+8\sqrt{x}+C. \end{aligned} \end{equation*}
    🔗
    🔗
    Be careful with products and quotients. In general, the antiderivative of a product is not the product of antiderivatives. When you can rewrite the expression as a sum of basic patterns, the problem becomes manageable.
    🔗
    🔗

    Subsection When You Must Rewrite First

    Example 4.1.10.

    Find a general antiderivative of \(f(x)=\sin^2(x)\text{.}\)
    🔗
    Solution.
    The function \(\sin^2(x)\) is not on our basic list. Use the identity \(\cos(2x)=1-2\sin^2(x)\text{,}\) which gives \(\sin^2(x)=\tfrac12(1-\cos(2x))\text{.}\) Now antidifferentiate:
    🔗
    \begin{equation*} \begin{aligned} \int \sin^2(x)\,dx &leftrightarrow \int \left(\tfrac12-\tfrac12\cos(2x)\right)\,dx\\ &\leftrightarrow \tfrac12 x-\tfrac12\cdot \tfrac12\sin(2x)+C\\ &= \tfrac12 x-\tfrac14\sin(2x)+C. \end{aligned} \end{equation*}
    🔗
    🔗
    🔗

    Subsection Acceleration, Velocity, and Position

    In motion problems, derivatives connect position, velocity, and acceleration: \(v(t)=s'(t)\) and \(a(t)=v'(t)=s''(t)\text{.}\) If we know acceleration and some initial conditions, we can recover velocity and position by antidifferentiating twice.
    🔗

    Example 4.1.11.

    A rocket has acceleration \(a(t)=t+1\) (in m/sec2). At time \(t=1\text{,}\) its height is \(s(1)=10\) meters and its velocity is \(v(1)=3\) m/sec. Find the velocity function \(v(t)\) and the height function \(s(t)\text{.}\)
    🔗
    Solution.
    Since \(v'(t)=a(t)=t+1\text{,}\) antidifferentiate:
    🔗
    \begin{equation*} v(t)=\tfrac12 t^2+t+C. \end{equation*}
    Use \(v(1)=3\text{:}\) \(\tfrac12(1)^2+1+C=3\text{,}\) so \(C=\tfrac32\text{.}\) Thus
    🔗
    \begin{equation*} v(t)=\tfrac12 t^2+t+\tfrac32. \end{equation*}
    Now antidifferentiate again because \(s'(t)=v(t)\text{:}\)
    🔗
    \begin{equation*} \begin{aligned} s(t) &= \int \left(\tfrac12 t^2+t+\tfrac32\right)\,dt \ \text{(as antiderivative facts)}\\ &= \tfrac12\cdot\tfrac13 t^3+\tfrac12 t^2+\tfrac32 t + B\\ &= \tfrac16 t^3+\tfrac12 t^2+\tfrac32 t + B. \end{aligned} \end{equation*}
    Use \(s(1)=10\text{:}\) \(\tfrac16+\tfrac12+\tfrac32+B=10\text{,}\) so \(B=\tfrac{47}{6}\text{.}\) Therefore
    🔗
    \begin{equation*} s(t)=\tfrac16 t^3+\tfrac12 t^2+\tfrac32 t+\tfrac{47}{6}. \end{equation*}
    With both conditions applied, the constants are determined and the solution is unique.
    🔗
