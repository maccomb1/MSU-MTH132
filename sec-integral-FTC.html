    Section 4.4 The Fundamental Theorem of Calculus

    Objectives

    • Explain how an integral with a variable upper limit defines an antiderivative.
      🔗
    • Use the First Fundamental Theorem of Calculus to differentiate integral functions.
      🔗
    • Use the Second Fundamental Theorem of Calculus to evaluate definite integrals from an antiderivative.
      🔗
    • Interpret \(\displaystyle \int_a^b f(x)\,dx\) as total change \(F(b)-F(a)\) when \(F'(x)=f(x)\text{.}\)
      🔗
    • Sketch an integral function \(\displaystyle I(x)=\int_a^x f(t)\,dt\) using the fact that \(I'(x)=f(x)\text{.}\)
      🔗
    🔗

    Subsection Integral as Antiderivative

    In the previous section we used a velocity function \(v(t)\) to compute distance traveled over a time interval \([a,b]\text{.}\) The distance is an integral defined as a limit of Riemann sums, adding up (velocity)\(\times\)(time) over many small time increments:
    🔗
    \begin{equation*} \text{distance traveled} \ =\ s(b)-s(a) \ =\ \lim_{\Delta t\to 0}\sum_{i=1}^n v(t_i)\,\Delta t \ =\ \int_a^b v(t)\,dt. \end{equation*}
    If we fix the starting time \(a\) and assume \(s(a)=0\text{,}\) then taking a variable endpoint \(b=x\) gives a function
    🔗
    \begin{equation*} s(x) \ =\ \int_a^x v(t)\,dt. \end{equation*}
    Since the rate of change of position is velocity, \(s'(x)=v(x)\text{.}\) The remarkable point is that the integral function \(\displaystyle \int_a^x v(t)\,dt\) always produces an antiderivative of \(v\text{,}\) even if we cannot find an antiderivative by reversing derivative formulas.
    🔗
    🔗

    Subsection The First Fundamental Theorem of Calculus

    Theorem 4.4.1. First Fundamental Theorem of Calculus.

    Let \(f\) be continuous on \([a,b]\) and define
    🔗
    \begin{equation*} I(x)\ =\ \int_a^x f(t)\,dt. \end{equation*}
    Then \(I'(x)=f(x)\) for \(x\in(a,b)\text{.}\) In particular, \(I\) is the unique antiderivative of \(f\) satisfying \(I(a)=0\text{.}\)
    🔗
    🔗
    In physical language: the rate of change of a cumulative effect up to time \(x\) is the strength of the effect at time \(x\text{.}\)
    🔗
    Proof idea. We start from the definition of the derivative: \(I'(x)=\lim_{h\to 0}\dfrac{I(x+h)-I(x)}{h}\text{.}\) Using the splitting rule for integrals, \(\int_a^{x+h}=\int_a^x+\int_x^{x+h}\text{,}\) so:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle \frac{I(x+h)-I(x)}{h} &=& \displaystyle \frac{1}{h}\left(\int_a^{x+h} f(t)\,dt-\int_a^x f(t)\,dt\right) \\[0.8em] &=& \displaystyle \frac{1}{h}\int_x^{x+h} f(t)\,dt. \end{array} \end{equation*}
    When \(h\) is small, the integral \(\int_x^{x+h} f(t)\,dt\) is the area under \(f\) over a very short interval, so it behaves like (height)\(\times\)(width). To make that precise, let \(N_h\) be the minimum value of \(f\) on \([x,x+h]\) and \(M_h\) be the maximum value on \([x,x+h]\text{.}\) Since \(N_h\le f(t)\le M_h\) on this interval, the bounds rule gives
    🔗
    \begin{equation*} \begin{array}{c} \displaystyle N_h \le \frac{1}{h}\int_x^{x+h} f(t)\,dt \le M_h. \end{array} \end{equation*}
    As \(h\to 0\text{,}\) the interval shrinks to the point \(x\text{,}\) so by continuity \(\lim_{h\to 0}N_h=\lim_{h\to 0}M_h=f(x)\text{.}\) By the squeeze theorem, \(\displaystyle \lim_{h\to 0}\frac{1}{h}\int_x^{x+h} f(t)\,dt=f(x)\text{,}\) hence \(I'(x)=f(x)\text{.}\) The condition \(I(a)=\int_a^a f(t)\,dt=0\) and the antiderivative uniqueness theorem give the uniqueness statement.
    🔗
    🔗

    Subsection Derivatives of Integral Functions

    The First Fundamental Theorem can be used as a basic derivative rule: if \(f\) is continuous, then
    🔗
    \begin{equation*} \frac{d}{dx}\left(\int_a^x f(t)\,dt\right) \ =\ f(x). \end{equation*}
    Here \(a\) is a constant, \(x\) is the input, and \(t\) is a dummy variable that only matters inside the integral. If the upper limit is a function \(g(x)\text{,}\) then the chain rule gives
    🔗
    \begin{equation*} \frac{d}{dx}\left(\int_a^{g(x)} f(t)\,dt\right) \ =\ f(g(x))\,g'(x). \end{equation*}

    Example 4.4.2.

    Find the derivative of \(\displaystyle F(x)=\int_{2x}^{x^3}\sin(t)\,dt\text{.}\)
    🔗
    Solution.
    Rewrite with a fixed lower limit and then differentiate:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle F(x) &=& \displaystyle \int_{2x}^{x^3}\sin(t)\,dt \ =\ \int_{0}^{x^3}\sin(t)\,dt - \int_{0}^{2x}\sin(t)\,dt. \end{array} \end{equation*}
    \begin{equation*} \begin{array}{rcl} \displaystyle F'(x) &=& \displaystyle \sin(x^3)\cdot (x^3)' \;-\; \sin(2x)\cdot (2x)' \\[0.6em] &=& 3x^2\sin(x^3)-2\sin(2x). \end{array} \end{equation*}
    🔗
    🔗

    Example 4.4.3.

    Let \(a\) and \(b\) be constants. Find \(F'(x)\) if \(\displaystyle F(x)=\int_{2a}^{b^3}\sin(t)\,dt\text{.}\)
    🔗
    Solution.
    The expression \(\int_{2a}^{b^3}\sin(t)\,dt\) is a number that does not depend on \(x\text{,}\) so \(F\) is constant and \(F'(x)=0\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Subsection Sketching Integral Functions

    An integral function \(I(x)=\int_a^x f(t)\,dt\) may not have an elementary formula. Even so, it can be understood: its derivative is \(I'(x)=f(x)\text{.}\) In other words, the slope of \(y=I(x)\) is determined by the height of \(y=f(x)\text{.}\) This lets us use the same curve-sketching ideas from earlier sections.
    🔗

    Example 4.4.4.

    Sketch the function \(\displaystyle I(x)=\int_0^x \sin(t^2)\,dt\text{.}\)
    🔗
    Solution.
    By the First Fundamental Theorem, \(I'(x)=\sin(x^2)\text{.}\) Critical points occur where \(I'(x)=0\text{,}\) that is where \(\sin(x^2)=0\text{.}\) This happens when \(x^2=k\pi\) for integers \(k\text{,}\) so \(x=0,\pm\sqrt{\pi},\pm\sqrt{2\pi},\ldots\text{.}\)
    🔗
    For concavity, \(I''(x)=(\sin(x^2))'=2x\cos(x^2)\text{.}\) So possible inflection points occur when \(2x\cos(x^2)=0\text{,}\) i.e. \(x=0\) or \(\cos(x^2)=0\text{,}\) which means \(x^2=\frac{\pi}{2}+k\pi\text{.}\)
    🔗
    Using the sign of \(I'(x)=\sin(x^2)\) gives the rough shape:
    🔗
    \begin{equation*} \begin{array}{c|c|c|c|c|c|c|c|c|c|c|c} x &\quad & -\sqrt{2\pi} &\quad & -\sqrt{\pi} &\quad & 0 &\quad & \sqrt{\pi} &\quad & \sqrt{2\pi} &\quad \\ \hline I'(x) & + & 0 & - & 0 & + & 0 & + & 0 & - & 0 & + \\ \hline I(x) & \nearrow & & \searrow & & \nearrow & 0 & \nearrow & & \searrow & & \nearrow \end{array} \end{equation*}
    Because \(\sin(x^2)\) is even, the integral function is odd: \(I(-x)=-I(x)\text{.}\) So the graph has \(180^\circ\) rotational symmetry about the origin.
    🔗
    🔗
    🔗
    🔗

    Subsection The Second Fundamental Theorem of Calculus

    Theorem 4.4.5. Second Fundamental Theorem of Calculus.

    Suppose \(F\) is an antiderivative of \(f\) on \([a,b]\text{,}\) so \(F'(x)=f(x)\text{.}\) Then
    🔗
    \begin{equation*} \int_a^b f(x)\,dx \ =\ F(b)-F(a). \end{equation*}
    That is, the integral of a rate of change is the total change.
    🔗
    🔗
    Proof idea. The First Fundamental Theorem says \(I(x)=\int_a^x f(t)\,dt\) is an antiderivative of \(f\text{.}\) Any two antiderivatives differ by a constant, so \(I(x)=F(x)+C\text{.}\) Since \(I(a)=0\text{,}\) we get \(0=F(a)+C\text{,}\) so \(C=-F(a)\text{.}\) Thus \(I(x)=F(x)-F(a)\text{,}\) and evaluating at \(x=b\) gives \(\int_a^b f(x)\,dx=I(b)=F(b)-F(a)\text{.}\)
    🔗

    Example 4.4.6.

    Evaluate \(\displaystyle \int_{-\sqrt{5}}^{\sqrt{5}} \left(5+4x^2-x^4\right)\,dx\text{.}\)
    🔗
    Solution.
    An antiderivative is \(F(x)=5x+\frac{4}{3}x^3-\frac{1}{5}x^5\text{.}\) Apply the theorem:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle \int_{-\sqrt{5}}^{\sqrt{5}} (5+4x^2-x^4)\,dx &=& F(\sqrt{5})-F(-\sqrt{5}) \\[0.6em] &=& 2F(\sqrt{5}) \\[0.6em] &=& 2\left(5\sqrt{5}+\frac{4}{3}(5\sqrt{5})-\frac{1}{5}(25\sqrt{5})\right) \\[0.6em] &=& 2\left(\frac{20}{3}\sqrt{5}\right) \ =\ \frac{40}{3}\sqrt{5}. \end{array} \end{equation*}
    🔗
    🔗

    Example 4.4.7.

    Determine the area under \(y=5+4x^2-x^4\) and above the \(x\)-axis.
    🔗
    Solution.
    First find where the graph meets the \(x\)-axis: \(5+4x^2-x^4=0\text{.}\) Let \(u=x^2\text{,}\) so the equation becomes \(5+4u-u^2=0\text{,}\) which has solutions \(u=-1\) and \(u=5\text{.}\) Only \(u=5\) gives real \(x\text{,}\) so the intercepts are \(x=\pm\sqrt{5}\text{.}\) The function is nonnegative between these intercepts, so the area is
    🔗
    \begin{equation*} \text{Area} \ =\ \int_{-\sqrt{5}}^{\sqrt{5}} (5+4x^2-x^4)\,dx \ =\ \frac{40}{3}\sqrt{5}. \end{equation*}
    Quick check: the base is about \(2\sqrt{5}\approx 4.47\) and the average height is around \(7\text{,}\) so an area near \(31\) is reasonable.
