    Section 1.4 Intermediate Value Theorem

    Objectives

    • State the Intermediate Value Theorem and explain in your own words what it guarantees about continuous functions.
      🔗
    • Apply the IVT to demonstrate that an equation has a solution within a given interval, even without finding the exact solution.
      🔗
    • Recognize situations where the IVT cannot be applied, such as when a function is not continuous on the entire interval.
      🔗
    • Interpret the IVT graphically and numerically, understanding how it ensures intermediate values are attained.
      🔗
    🔗
    One of the most important applications of continuity is the Intermediate Value Theorem (IVT). This theorem provides a way to guarantee the existence of solutions to equations, even when we cannot solve them exactly. It also explains why continuous functions must pass through all intermediate values between two function outputs.
    🔗

    Theorem 1.4.1. Intermediate Value Theorem (IVT).

    If \(f(x)\) is continuous for \(x\) in the interval \([a,b]\text{,}\) and \(r\) is between \(f(a)\) and \(f(b)\text{,}\) meaning either \(f(a) < r < f(b)\) or \(f(a) > r > f(b)\text{,}\) then there is at least one (but possibly more) value of \(c \in (a,b)\)
     1 
    Here \(c\) lies in the open interval \((a,b)\text{,}\) between \(a\) and \(b\) but excluding both endpoints: \(a<c<b\text{.}\)
    such that \(f(c)=r\text{.}\)
    🔗
    🔗

    Remark 1.4.3.

    This says that as the function value \(f(x)\) goes continuously from \(f(a)\) to \(f(b)\text{,}\) perhaps rising and falling many times, it must pass through every value \(r\) between \(f(a)\) and \(f(b)\text{.}\)
    🔗
    🔗
    Let’s see how the theorem works in practice.
    🔗

    Example 1.4.4. Applying the IVT.

    Let \(f(x)=x^3+x-1\text{.}\) Show that there exists a number \(c\in(0,1)\) such that \(f(c)=0\text{.}\)
    🔗
    Solution.
    The function \(f(x)=x^3+x-1\) is a polynomial, hence continuous on every interval, in particular on \([0,1]\text{.}\)
    🔗
    Evaluate the endpoints: \(f(0)=0^3+0-1=-1\) and \(f(1)=1^3+1-1=1\text{.}\) Thus \(f(0)=-1<0<1=f(1)\text{,}\) so the value \(0\) lies between \(f(0)\) and \(f(1)\text{.}\)
    🔗
    By the Intermediate Value Theorem, there exists \(c\in(0,1)\) such that \(f(c)=0\text{.}\) So although we do not know the exact value of \(c\) we know that is must exist and what interval it is in.
    🔗
    🔗
    🔗
    In the previous example, everything worked out neatly: the function was continuous on the interval and the target value fell between the two endpoint values. But if continuity is broken, the Intermediate Value Theorem no longer applies. The next example illustrates this.
    🔗

    Example 1.4.5.

    Consider the function: \(\quad f(x) = \dfrac{x-1}{x-5}\)
    🔗
    1. Does \(f(x)\) take on the value \(2\) somewhere in the interval \([1,6]\text{?}\)
      🔗
    2. Can the Intermediate Value Theorem be applied to justify your answer to (a)? Why or why not?
      🔗
    Solution.
    1. Solve \(f(x)=2\text{:}\)
      🔗
      🔗
      \(\begin{align*} \dfrac{x-1}{x-5} \amp = 2 \\ \Longrightarrow\quad x-1 \amp = 2(x-5) \\ \Longrightarrow\quad x-1 \amp = 2x-10 \\ x \amp = 9. \end{align*}\)
      Since \(9 \notin [1,6]\text{,}\) the equation \(f(x)=2\) has no solution in \([1,6]\text{.}\) So No, \(f(x)\) does not take the value \(2\) on \([1,6]\text{.}\)
      🔗
      🔗
    2. No — the Intermediate Value Theorem (IVT) requires continuity on a closed interval. The function \(f\) is not continuous on \([1,6]\) because it has a vertical asymptote at \(x=5\) (the denominator is zero there).
      🔗
      So even though \(f(1)=0\) and \(f(6)=5\text{,}\) and the value \(2\) lies between \(0\) and \(5\text{,}\) because \(f(x)\) is not continuous on the entire interval \([1,6]\) the Intermediate Value Theorem cannot be applied.
      🔗
      🔗
    🔗
    🔗
    Finally, let’s look at a famous equation where the IVT guarantees the existence of a solution, even though we cannot solve it with algebra alone.
    🔗

    Example 1.4.6.

    Show that there exists a solution \(x=c\) to the equation \(\cos(x)=x\text{.}\)
    🔗
    Solution.
    There is no simple algebraic way to solve \(\cos(x)=x\) directly. Instead, define a new function
    🔗
    \begin{equation*} f(x) = \cos(x) - x. \end{equation*}
    Then solving \(\cos(x)=x\) is equivalent to finding a root of \(f(x)\text{,}\) that is, some \(c\) with \(f(c)=0\text{.}\)
    🔗
    First check some values of \(f(x)\) on the interval \([0,\pi]\text{:}\)
    🔗
    • \(f(0) = \cos(0) - 0 = 1 - 0 = 1 > 0\text{.}\)
      🔗
    • \(f(\pi) = \cos(\pi) - \pi = -1 - \pi < 0\text{.}\)
      🔗
    Thus, \(f(0) > 0\) and \(f(\pi) < 0\text{,}\) so the function values change sign between \(x=0\) and \(x=\pi\text{.}\)
    🔗
    Next, recall that \(f(x) = \cos(x) - x\) is continuous: the cosine function is continuous everywhere, and \(x\) is also continuous, so their difference is continuous.
    🔗
    By the Intermediate Value Theorem, if a continuous function takes on values of opposite signs at the endpoints of an interval, then it must equal zero for some number between those endpoints. Therefore, there exists \(c \in (0,\pi)\) such that \(f(c)=0\text{.}\)
    🔗
    In other words, there is some number \(c\) with \(\cos(c) = c\text{.}\) (In fact, numerically \(c \approx 0.739\text{,}\) but the IVT only guarantees existence, not the exact value.)
    🔗
    🔗
    🔗
    The Intermediate Value Theorem is powerful because it guarantees the existence of solutions even when we cannot compute them exactly. Without needing advanced tools or calculators, we can still be certain that roots and crossing points must lie within specific intervals — giving us both confidence and direction in solving problems that might otherwise feel impossible.
    🔗

    Exercises Practice / Study Problems

    Some True/False Questions.

    Determine whether each statement is true or false and explain your reasoning
    🔗
    1.
    The IVT can be used to show that \(x^3+2x-5=0\) has a solution in the interval \([1,2]\text{.}\)
    🔗
    Solution.
    True. Let \(f(x)=x^3+2x-5\) (a polynomial, hence continuous). Then \(f(1)=1+2-5=-2<0\) and \(f(2)=8+4-5=7>0\text{.}\) Since 0 lies between \(f(1)\) and \(f(2)\text{,}\) the IVT guarantees a root \(c\in(1,2)\text{.}\)
    🔗
    🔗
    🔗
    2.
    A function \(f\) is continuous on the interval \([1,5]\text{.}\) Values of \(f(x)\) are given in the table below:
    🔗
    \begin{equation*} \begin{array}{|c|c|} \hline x \amp f(x) \\ \hline 1 \amp -3 \\ 2 \amp -1 \\ 3 \amp 4 \\ 4 \amp 7 \\ 5 \amp 10 \\ \hline \end{array} \end{equation*}
    There must exist some \(c \in [1,3]\) such that \(f(c) = 3\text{.}\)
    🔗
    Solution.
    True. Since \(f(1) = -3\) and \(f(3) = 4\text{,}\) and \(f\) is continuous, the IVT guarantees a value \(c \in (1,3)\) with \(f(c)=3\text{.}\)
    🔗
    🔗
    🔗
    3.
    The IVT tells us exactly where a solution occurs and gives us the formula for the solution.
    🔗
    Solution.
    False. The IVT guarantees existence of at least one solution in an interval but does not locate it exactly or provide a formula.
    🔗
    🔗
    🔗
    4.
    If \(f\) is continuous on \([0,2]\) and \(f(0)=-1\text{,}\) \(f(2)=3\text{,}\) then \(f(x)=0\) for some \(x \in (0,2)\text{.}\)
    🔗
    Solution.
    True. Continuity on \([0,2]\) and opposite signs at the endpoints imply, by IVT, that some \(c\in(0,2)\) satisfies \(f(c)=0\text{.}\)
    🔗
    🔗

    Multiple Choice.

    For each question, choose the single best answer from the options provided.
    🔗
    5.
    Which of the following best describes what the IVT guarantees?
    🔗
    1. The exact solution of \(f(x)=r\text{.}\)
      🔗
    2. The existence of at least one solution to \(f(x)=r\) in \((a,b)\text{,}\) if \(f\) is continuous.
      🔗
    3. The maximum value of \(f\) on \([a,b]\text{.}\)
      🔗
    4. The slope of the tangent line to \(f\text{.}\)
      🔗
    Solution.
    Answer: B. The IVT guarantees existence (not exact location or value) provided continuity on \([a,b]\) and that \(r\) lies between \(f(a)\) and \(f(b)\text{.}\)
    🔗
    🔗
    🔗
    6.
    Let \(f(x)=x^2-2\text{.}\) On which interval can the IVT be used to show a solution to \(f(x)=0\) exists?
    🔗
    1. \(\displaystyle [0,1]\)
      🔗
    2. \(\displaystyle [1,2]\)
      🔗
    3. \(\displaystyle [2,3]\)
      🔗
    4. \(\displaystyle [-3,-2]\)
      🔗
    Solution.
    Answer: B. \(f(1)=-1<0\text{,}\) \(f(2)=2>0\) so a root lies in \((1,2)\text{.}\) The other intervals have no sign change.
    🔗
    🔗
    🔗
    7.
    The IVT can be used to show that \(\cos(x)=x\) has a solution in which interval?
    🔗
    1. \(\displaystyle [0,1]\)
      🔗
    2. \(\displaystyle [1,2]\)
      🔗
    3. \(\displaystyle [2,3]\)
      🔗
    4. \(\displaystyle [3,4]\)
      🔗
    Solution.
    Answer: A. For \(f(x)=\cos x - x\text{,}\) \(f(0)=1>0\) and \(f(1)=\cos(1)-1<0\text{;}\) continuity gives a root in \((0,1)\text{.}\)
    🔗
    🔗
    🔗
    8.
    Which of the following is not an application of the IVT?
    🔗
    1. Proving an equation has a solution in an interval.
      🔗
    2. Guaranteeing that a continuous function crosses a certain horizontal line.
      🔗
    3. Finding the exact numerical value of a solution.
      🔗
    4. Explaining why a temperature must have reached 70°F if it rose from 65°F to 75°F.
      🔗
    Solution.
    Answer: C. IVT does not give exact values; it guarantees existence.

    Written Response.

    Answer each question in complete sentences, showing your reasoning and any supporting work.
    🔗
    9.
    Use the IVT to prove that the equation \(x^3-7x+5=0\) has a solution in the interval \((0,2)\text{.}\)
    🔗
    Solution.
    Let \(f(x)=x^3-7x+5\) (continuous). Compute \(f(0)=5>0\) and \(f(2)=8-14+5=-1<0\text{.}\) Since 0 lies between these values, the IVT guarantees a root \(c\in(0,2)\text{.}\)
    🔗
    🔗
    🔗
    10.
    Explain why the function \(f(x)=\dfrac{1}{x}\) does not satisfy the conditions of the IVT on \([-1,1]\text{.}\)
    🔗
    Solution.
    \(f\) is not defined at \(x=0\) and hence not continuous on the closed interval \([-1,1]\text{.}\) The IVT requires continuity on the entire interval.
    🔗
    🔗
    🔗
    11.
    A function \(f\) is continuous on \([0,5]\text{,}\) with \(f(0)=-2\) and \(f(5)=7\text{.}\) Use the IVT to explain why \(f(x)=3\) must have a solution in \((0,5)\text{.}\)
    🔗
    Solution.
    Since \(3\) lies between \(-2\) and \(7\text{,}\) and \(f\) is continuous on \([0,5]\text{,}\) the IVT guarantees \(c\in(0,5)\) with \(f(c)=3\text{.}\)
    🔗
    🔗
    🔗
    12.
    Suppose a car’s speed (a continuous function of time) was 40 mph at 2:00 pm and 65 mph at 2:10 pm. Explain, using the IVT, why there must have been a time when the car’s speed was exactly 55 mph.
    🔗
    Solution.
    Let \(v(t)\) be speed, continuous on \([2{:}00,2{:}10]\text{.}\) Since \(55\) lies between \(40\) and \(65\text{,}\) the IVT implies some time \(c\) in that interval with \(v(c)=55\text{.}\)
    🔗
    🔗
    🔗
    13.
    Why does the IVT not apply to the function \(f(x)=\dfrac{x^2-1}{x-1}\) on \([0,2]\text{,}\) even though \(f(0)=1\) and \(f(2)=3\text{?}\)
    🔗
    Solution.
    The given formula is undefined at \(x=1\) (removable discontinuity), so the function is not continuous on the entire closed interval \([0,2]\text{.}\) Thus the hypotheses of the IVT are not met.
