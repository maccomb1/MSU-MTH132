    Section 1.1 Limits from Graphs and Notation

    Objectives

    • Understand the intuitive definition of a limit and how to express it using mathematical notation.
      🔗
    • Determine limits from tables of values and from the graph of a function.
      🔗
    • Recognize when a limit exists, and identify cases where the limit does not exist.
      🔗
    • Interpret and compute one-sided limits, and explain how they relate to the existence of two-sided limits.
      🔗
    • Understand the meaning of infinite limits and how they relate to vertical asymptotes.
      🔗
    • Locate vertical asymptotes and describe a function’s behavior on both sides of each asymptote.
      🔗
    🔗

    Subsection 1.1.1 Definition of Limits

    The concept of a limit is one of the cornerstones of calculus. Limits allow us to describe precisely what happens to a function’s values as the input approaches a particular number—whether or not the function is even defined at that point. They provide the language for defining continuity, derivatives, and integrals, and they help us capture “approaching” behavior that can’t be expressed by direct substitution alone. Before moving on to more formal ideas, we start with an intuitive definition of a limit.
    🔗

    Definition 1.1.1. Limit (intuitive).

    Consider a function \(f(x)\) and numbers \(L\text{,}\) \(a\text{.}\) Then the limit of \(f(x)\) equals \(L\) as \(x\) approaches \(a\text{,}\) in symbols \(\displaystyle\lim_{x\to a} f(x)=L\text{,}\) whenever \(f(x)\) can be forced arbitrarily close to \(L\) by making \(x\) sufficiently close to (but not equal to) \(a\text{.}\) If we cannot make \(f(x)\) as close to a single value as we would like as \(x\) approaches \(a\text{,}\) then we say that \(f\) does not have a limit as \(x\) approaches \(a\text{.}\)
    🔗
    🔗
    That is, \(f(x)\) approximates \(L\) to within any desired error tolerance, for all values of \(x\) within some small distance from \(a\) (but \(x \neq a\)). One more way to say it: if we make a table of \(f(x)\) for any sample values of \(x\) getting closer and closer to \(a\) (such as \(x = a+0.1\text{,}\) \(a+0.01\text{,}\) etc.), then the values of \(f(x)\) will get as close as we like to \(L\) (though they might never quite reach \(L\)).

    Subsection 1.1.2 Evaluating Limits

    Let’s make sure the definition makes sense. Here is a nice first problem to attempt.
    🔗

    Example 1.1.3. Our First Limit.

    Consider the function \(f(x) = \dfrac{x^3-2x^2+x-2}{x-2}\)
    1. Using a calculator, evaluate the function at the indicated \(x\)-values and fill in the corresponding entries in the table.

      🔗
    2. Based on the intuitive limit definition guess the limit value \(\displaystyle\lim_{x\to 2} \dfrac{x^3-2x^2+x-2}{x-2}\)
      🔗
    3. View the graph of \(f(x)\) in Desmos. Use words to explain to another student how the table relates to the graph and how the limit can be determined from the graph.

      🔗
    2. \(\displaystyle \displaystyle\lim_{x\to 2} \dfrac{x^3-2x^2+x-2}{x-2} = 5\)
      🔗
    3. As \(x\) approaches \(2\) from either the left or the right, the output values \(f(x)\) get closer and closer to \(5\text{.}\) Although \(f(2)\) is undefined (there is a hole in the graph), the limit still exists and equals \(5\text{.}\)

    Some limits are easy because we can plug in \(x = a\) to get the limiting value \(\displaystyle\lim_{x \to a} f(x) = f(a)\text{,}\) in which case we say \(f(x)\) is continuous at \(x = a\text{.}\) Graphically, as in the above figure 1.1.2, this means the curve has no jump or hole at \((a, f(a))\text{.}\)

    Example 1.1.4. A Nice Continuous Example.

    Evaluate \(\displaystyle \lim_{x\to 5} x^2 \text{.}\)
    🔗
    Solution.
    Full Written Solution:
    🔗
    Since our function \(f(x) = x^2\) is continuous around \(x = 5\) (we can draw the parabola without lifting our pencil), we can simply plug \(x = 5\) into \(f(x)\) to get the answer \(\displaystyle \lim_{x \to 5} x^2 = 25\text{.}\)
    🔗
    More Rigorous Solution:
    🔗
    Algebraically, if \(x\) is close enough to \(5\text{,}\) say \(x = 5 + h\) for some small \(h\) (close to \(0\)), then
    \begin{equation*} x^2 = (5 + h)^2 = 5^2 + 2(5h) + h^2 = 25 + 10h + h^2 \end{equation*}
    which is forced as close as we like to \(L = 25\) if \(h\) is small enough to 0 (either positive or negative).
    🔗
    🔗
    🔗
    We have seen a few problems where a limit exists, but in fact, as the 2004 movie Mean Girls teaches us, sometimes
    🔗
    One way this can occur is if \(f(x)\) does not approach a single limiting value as \(x\) approaches \(a\text{.}\) In such cases, we declare that the limit does not exist, and the symbol \(\displaystyle\lim_{x \to a} f(x)\) has no meaning. Now let’s see such an example.
    🔗

    Example 1.1.6. When a limit does not exist.

    Consider the sign function, defined by
    \begin{equation*} \text{sgn}(x) = \dfrac x{|x|} = \left\{\begin{array}{ccl} +1 \amp \text{ for $x \gt 0$}\\ -1 \amp \text{ for $x \lt 0$} \end{array}\right. \end{equation*}
    🔗
    Which when graphed looks like:
    🔗
    Determine the limit \(\displaystyle\lim_{x\to 0} \text{sgn}(x)\)
    🔗
    Solution.
    Near \(x = 0\text{,}\) the function cannot be forced close to any single output value. That is, \(\displaystyle\lim_{x \to 0} \operatorname{sgn}(x) \neq 1\text{,}\) since no matter how close \(x\) gets to \(0\text{,}\) there are some \(x\) (namely negative) for which \(\operatorname{sgn}(x)\) is far from \(1\text{;}\) and similarly \(\displaystyle\lim_{x \to 0} \operatorname{sgn}(x)\) is not \(-1\text{,}\) nor \(0\text{,}\) nor any other value. In such cases, it is most appropriate to simply say \(\displaystyle\lim_{x \to 0} \operatorname{sgn}(x)\) does not exist.
    🔗
    🔗
    🔗

    Remark 1.1.8.

    An important feature of \(\displaystyle \lim_{x \to a} f(x)\) is that it does not depend on \(f(a)\text{,}\) even if \(f(a)\) is undefined: the limit only notices values of \(f(x)\) for \(x \neq a\text{.}\) As a quick mini example, define \(g(x) = 1\) for \(x \neq 3\text{,}\) and \(g(3) = 2\text{,}\) having the graph:
    🔗
    Then \(\displaystyle \lim_{x \to 3} g(x) = 1\text{,}\) since if \(x\) is close enough to (but unequal to) \(3\text{,}\) then \(g(x)\) is arbitrarily close to \(L = 1\) (in fact \(g(x) = L\)). Again, \(\displaystyle \lim_{x \to 3} g(x) \neq g(3) = 2\text{,}\) and \(g(x)\) is not continuous at \(x = 3\text{.}\)
    🔗
    🔗

    Subsection 1.1.3 One-sided Limits

    Sometimes, the behavior of a function as you approach a point depends on which direction you come from. For example, a graph might approach one value when moving in from the left, but a completely different value when coming in from the right like in Example 1.1.6.
    🔗
    And so, let’s define another type of limit:
    🔗

    Definition 1.1.10. One-Sided Limits.

    One-sided limits (from the right or left) consider only values of \(x\) on one side of \(a\text{.}\) The limit of \(f(x)\) equals \(L\) as \(x\) approaches \(a\) from the right, denoted \(\displaystyle\lim_{x\to a^{+}} f(x) = L\text{,}\) whenever \(f(x)\) can be made arbitrarily close to \(L\) by taking \(x\) sufficiently close to (but greater than) \(a\text{.}\) The limit from the left, denoted \(\displaystyle\lim_{x\to a^{-}} f(x) = L\text{,}\) is defined similarly, except with \(x\) less than \(a\text{.}\)
    🔗
    🔗

    Remark 1.1.11.

    If we have the ordinary limit \(\displaystyle\lim_{x\to a} f(x) = L\text{,}\) then the left and right limits must both have the same value \(L\text{.}\) In the Example 1.1.4 above, we have \(\displaystyle\lim_{x\to 5^+} x^2 = \lim_{x\to 5^-} x^2 = 5^2\text{.}\)
    🔗
    However in Example 1.1.6 we have, \(\displaystyle\lim_{x\to 0^+} \operatorname{sgn}(x) = 1\) and \(\lim_{x\to 0^-} \operatorname{sgn}(x) = -1\text{,}\) even though \(\displaystyle\lim_{x\to 0} \operatorname{sgn}(x)\) does not exist.
    🔗
    🔗
    In fact, we can make our first Theorem from this idea
    🔗

    Theorem 1.1.12.

    For a function \(f\) and real number \(a\text{,}\) the (two-sided) limit exists if and only if both one-sided limits exist and are equal. Precisely,
    🔗
    \(\displaystyle \lim_{x\to a} f(x) = L\) if and only if \(\left(\displaystyle \lim_{x\to a^-} f(x) = L \;\text{ and }\; \displaystyle \lim_{x\to a^+} f(x) = L\right).\)
    🔗
    🔗

    Proof.

    A full proof of this theorem requires the formal \(\varepsilon\)–\(\delta\) definition of a limit and is beyond the scope of this course. You will encounter this level of rigor in MTH 320, where you will learn how to construct proofs of results like this one from first principles.
    🔗
    🔗
    🔗

    Subsection 1.1.4 Infinite Limits

    Up to this point, most of the limits we’ve encountered have approached a finite number. However, some functions behave very differently: as you get closer to certain \(x\)-values, the function’s values grow without bound—either shooting up toward positive infinity or diving down toward negative infinity. In these cases, the function doesn’t have a finite limit, but the way it “blows up” is still predictable.
    🔗

    Definition 1.1.13. Infinite Limits and Vertical Asymptotes.

    We define infinite limits as follows: \(\displaystyle\lim_{x\to a} f(x) = \infty\) means that \(f(x)\) can be made larger than any real number (for example, \(f(x) > 1000\)) by taking \(x\) sufficiently close to (but not equal to) \(a\text{.}\)
    🔗
    Similarly \(\displaystyle\lim_{x\to a} f(x) = -\infty\) means that \(f(x)\) can be made smaller than any real number (for example, \(f(x) < -9547\)) by taking \(x\) sufficiently close to (but not equal to) \(a\text{.}\)
    🔗
    Such behavior signals a vertical asymptote at \(x=a\text{.}\)
    🔗
    🔗

    Remark 1.1.14.

    The symbol \(\infty\) (and likewise \(-\infty\)) has no meaning by itself; it is simply a way of saying that \(f(x)\) can be made as large as desired.
    🔗
    🔗

    Example 1.1.15.

    Consider the function \(f(x) = \dfrac{1}{|x|} \text{.}\)
    🔗
    Which when graphed looks like:
    🔗
    Answer the following questions.
    🔗
    1. Using the graph of \(f(x)\) evaluate \(\displaystyle\lim_{x\to 0} f(x)\) and explain your reasoning.
      🔗
    2. Try now without the graph (using only the equation of \(f(x)\)) to evaluate \(\displaystyle\lim_{x\to 0} f(x)\) and explain your reasoning.
      🔗
    Solution.
    1. Looking at the graph \(y = \dfrac{1}{|x|}\text{,}\) we can notice two key features near \(x = 0\text{:}\)
      🔗
      • The graph is in two pieces — one in quadrant II (for \(x < 0\)) and one in quadrant I (for \(x > 0\)).
        🔗
      • Both pieces shoot straight up as they get close to \(x = 0\text{.}\)
        🔗
      This means the function grows beyond any large number we can choose when \(x\) is close enough to \(0\text{.}\) And so we know \(\displaystyle\lim_{x\to 0} f(x)=\infty\)
      🔗
      🔗
    2. As \(x\) gets very close to \(0\text{,}\) \(|x|\) gets very small — like \(0.1\text{,}\) \(0.01\text{,}\) \(0.001\text{,}\) and so on. Dividing \(1\) by a very small positive number gives a very large positive number (for example, \(\frac{1}{0.1} = 10\text{,}\) \(\frac{1}{0.01} = 100\text{,}\) \(\frac{1}{0.001} = 1000\)). In general, we can shorthand this fact as \(\left(\frac 1{\text{tiny}}=\text{huge}\right)\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Problem 1.1.17.

    True or False: \(\displaystyle \lim_{x \to 0} \frac{1}{x} = \infty\text{.}\)
    🔗
    Solution.
    False. The two one-sided limits behave differently: \(\displaystyle \lim_{x \to 0^+} \frac{1}{x} = \infty\) but \(\displaystyle \lim_{x \to 0^-} \frac{1}{x} = -\infty\text{.}\) Because the left and right limits are not equal, the two-sided limit \(\displaystyle \lim_{x \to 0} \frac{1}{x}\) does not exist.
    🔗
    🔗
    🔗
    Note: Be careful to read problems in this class carefully! Here is problem that is quite similar looking but has a different result.
    🔗

    Problem 1.1.18.

    True or False: \(f(x) = \dfrac{1}{x}\) has a vertical asymptote at \(x = 0\text{.}\)
    🔗
    Solution.
    True. As \(x\) approaches \(0\) from the right, \(\dfrac{1}{x} \to \infty\text{,}\) and as \(x\) approaches \(0\) from the left, \(\dfrac{1}{x} \to -\infty\text{.}\) This unbounded behavior around \(x=0\) means the line \(x=0\) is a vertical asymptote.
    🔗
    🔗
    🔗
    🔗

    Subsection 1.1.5 Locating Vertical Asymptotes

    A vertical asymptote occurs at a value of \(x\) where a function grows without bound (positively or negatively) as \(x\) approaches that value. This often happens when the function’s formula has a denominator that becomes zero while the numerator stays nonzero, making the function’s value extremely large in magnitude near that point.
    🔗

    Strategy 1.1.19. How to Locate Vertical Asymptotes.

    To locate possible vertical asymptotes, we look for values of \(x\) that make the denominator zero but do not also make the numerator zero. Once we have these candidates, we check the one-sided limits to determine whether the function increases toward \(\infty\) or decreases toward \(-\infty\) on each side.
    🔗
    🔗
    Now let’s see it in action with an example
    🔗

    Example 1.1.20.

    Consider the function \(\displaystyle f(x)=\dfrac{2x-4}{x^2-4x+3}\text{.}\)
    🔗
    1. Locate the vertical asymptotes for \(f(x)\text{.}\)
      🔗
    2. For each vertical asymptote, describe the behavior of \(f(x)\) as \(x\) approaches from both the left and the right.
      🔗
    Solution.
    Check Your Answers:
    🔗
    1. The vertical asymptotes are at \(x=1\) and \(x=3\text{.}\)
      🔗
    2. Here is the function’s behavior around each asymptote:
      🔗
      \begin{equation*} \begin{array}{rlrl} \displaystyle\lim_{x\to 1^-} f(x) \amp = -\infty \amp \displaystyle\lim_{x\to 3^-} f(x) \amp = -\infty \\ \displaystyle\lim_{x\to 1^+} f(x) \amp = \infty \amp \displaystyle\lim_{x\to 3^+} f(x) \amp = \infty \end{array} \end{equation*}
      🔗
    Step by Step Video Solution:
    🔗
    🔗
    🔗
    🔗
    And lastly, a true/false problem on the same subject
    🔗

    Problem 1.1.21.

    True or False: The function \(\displaystyle f(x) = \frac{2x - 2}{x^{2} - 4x + 3} \) has a vertical asymptote at \(x = 1\text{.}\)
    🔗
    Solution.
    False. Factoring numerator and denominator we get
    \begin{equation*} f(x) \;=\; \frac{2(x-1)}{(x-1)(x-3)} , \end{equation*}
    Notice that if we try to naively plug in \(x=1\) we get 0 in both the numerator and denominator. From Remark 1.1.19 we should suspect that this will not yield a vertical asymptote.
    🔗
    To verify we can plug in numbers close to \(x=1\) on either side to see what is happening to \(f(x)\text{.}\)
    🔗
    \begin{equation*} \begin{array}{rl} f(0.99) \amp \approx -0.995\\ f(1.01) \amp \approx -1.005 \end{array} \end{equation*}
    From this we suspect \(\displaystyle \lim_{x\to 1}f(x)= -1\text{.}\) And in fact if we simplify
    \begin{equation*} f(x) = \dfrac{2(x-1)}{(x-1)(x-3)} = \dfrac{2}{x-3} \end{equation*}
    and then plug in \(x=1\) into the simplified function we get \(\dfrac{2}{1-3} = -1\text{.}\) We will learn in the next section that this will turn out to be a good strategy for evaluating limits. And so, \(x=1\) does not yield a vertical asymptote.
    🔗

    Exercises 1.1.6 Practice / Study Problems

    Some True/False Problems.

    Determine weather each statement is true or false and explain your reasoning
    🔗
    1.
    True or False: If \(\displaystyle \lim_{x\to 4} f(x) = 7\text{,}\) then \(f(4)=7\text{.}\)
    🔗
    Solution.
    False. A limit describes nearby behavior; the function value at the point can differ (or be undefined).
    🔗
    🔗
    🔗
    2.
    True or False: If \(\displaystyle \lim_{x\to 2^-} f(x) = 3\) and \(\displaystyle \lim_{x\to 2^+} f(x) = 3\text{,}\) then \(\displaystyle \lim_{x\to 2} f(x) = 3\text{.}\)
    🔗
    Solution.
    True. Equality of both one-sided limits guarantees the two-sided limit exists and has that value.
    🔗
    🔗
    🔗
    3.
    True or False: If \(\displaystyle \lim_{x\to a} f(x) = \infty\text{,}\) then \(f(a)\) is undefined.
    🔗
    Solution.
    False. The limit concerns values near \(a\text{;}\) \(f(a)\) may be anything (defined or not) without affecting the infinite limit.
    🔗
    🔗
    🔗
    4.
    True or False: If \(\displaystyle \lim_{x\to a} f(x)\) does not exist, then both \(\displaystyle \lim_{x\to a^-} f(x)\) and \(\displaystyle \lim_{x\to a^+} f(x)\) do not exist.
    🔗
    Solution.
    False. One-sided limits may exist but differ; that difference causes the two-sided limit to fail.
    🔗
    🔗
    🔗
    5.
    True or False: If \(x=3\) is a vertical asymptote, then at least one of \(\displaystyle \lim_{x\to 3^-} f(x)\) or \(\displaystyle \lim_{x\to 3^+} f(x)\) is infinite.
    🔗
    Solution.
    True. A vertical asymptote is characterized by unbounded behavior from one or both sides.
    🔗
    🔗
    🔗
    6.
    True or False: If the graph decreases without bound as \(x\to 1^-\text{,}\) then \(\displaystyle \lim_{x\to 1^-} f(x) = -\infty\text{.}\)
    🔗
    Solution.
    True. “Decreases without bound” means the values become arbitrarily negative.
    🔗
    🔗
    🔗
    7.
    True or False: \(\displaystyle \lim_{x\to 0} \frac{1}{x^2} = \infty\text{.}\)
    🔗
    Solution.
    True. Values grow beyond any bound from either side since \(x^2>0\) for \(x\neq 0\text{.}\)
    🔗
    🔗
    🔗
    8.
    True or False: If \(f(x)=\dfrac{2x}{x^2-9}\text{,}\) then \(x=3\) is a vertical asymptote.
    🔗
    Solution.
    True. \(x^2-9=(x-3)(x+3)\text{;}\) the numerator is nonzero at \(x=3\text{,}\) so \(x=3\) is a VA.
    🔗
    🔗
    🔗
    9.
    True or False: If a rational function’s denominator is zero at \(x=a\text{,}\) then the function must have a vertical asymptote at \(x=a\text{.}\)
    🔗
    Solution.
    False. A common factor in numerator and denominator could cancel instead like in Problem 1.1.21.
    🔗
