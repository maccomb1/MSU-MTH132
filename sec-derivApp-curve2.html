    Section 3.7 Curve Sketching Part 2

    Objectives

    • Apply the Second Derivative Test to classify critical points as local maxima, local minima, or inconclusive.
      🔗
    • Determine if a function has even and odd symmetry and use symmetry to reduce the amount of work required in a sketch.
      🔗
    • Use the full curve sketching method to analyze and sketch a variety of functions including:
      • Identifying domain, asymptotes, intercepts, and symmetry.
        🔗
      • Find and classify critical points and determine where the function increases or decreases using \(f'(x)\text{.}\)
        🔗
      • Find inflection points and determine intervals of concavity using \(f''(x)\text{.}\)
        🔗
      • Combine all information to produce a clear sketch of the graph of the function.
        🔗
      🔗
      🔗
    🔗
    At this point, you may be asking, "Why do I need to learn to sketch graphs by hand when there are computers and calculators that can graph instantly?" One reason is that calculus helps us focus on the important parts of the graph: critical points, asymptotes, and regions where behavior changes. Another reason is error checking. A quick sketch, combined with a few well-chosen test values, can reveal when a result is unreasonable long before a mistake spreads into a larger calculation. Finally, sketching by hand is a skill that can be useful in many contexts, such as when you are explaining a graph to someone else, or when you are trying to understand a graph that is already drawn.
    🔗
    So in this section, we first learn a few more skills to aid our curve sketching abilities. Then we will work through several longer problems that apply the full curve sketching method.
    🔗

    Subsection The Second Derivative Test

    The First Derivative Test classifies critical points by checking whether \(f'(x)\) changes sign. When \(f''(c)\) exists, there is a quicker classification that often works.
    🔗

    Theorem 3.7.1. Second Derivative Test.

    Let \(f\) be twice differentiable near \(c\text{,}\) and suppose \(f'(c)=0\text{.}\)
    🔗
    • If \(f''(c)>0\text{,}\) then \(f\) has a local minimum at \(x=c\text{.}\)
      🔗
      🔗
    • If \(f''(c)<0\text{,}\) then \(f\) has a local maximum at \(x=c\text{.}\)
      🔗
      🔗
    • If \(f''(c)=0\text{,}\) then the test is inconclusive.
      🔗
      🔗
    🔗

    Remark 3.7.2.

    If \(f''(c)=0\text{,}\) the point could still be a local maximum, a local minimum, or neither. In that case, you should return to the First Derivative Test (or another method).
    🔗
    🔗

    Example 3.7.3.

    Use the Second Derivative Test to classify the critical points of \(f(x)=x^3-3x^2+4\text{.}\)
    🔗
    Solution.
    Doing some prep work, the derivatives are \(f'(x)=3x^2-6x\) and \(f''(x)=6x-6\text{.}\) Now to find the critical points, we set \(f'(x)=0\text{,}\) which gives \(3x^2-6x=0\text{.}\) Factoring gives \(3x(x-2)=0\text{,}\) so the critical points are \(x=0\) and \(x=2\text{.}\) Note that \(f'(x)\) is defined everywhere, so there are no critical points from undefined derivatives.
    🔗
    Evaluating the second derivative at these points gives \(f''(0)=-6<0\text{,}\) so there is a local maximum at \(x=0\text{,}\) and \(f''(2)=6>0\text{,}\) so there is a local minimum at \(x=2\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Subsection Symmetry

    Sometimes symmetry can dramatically reduce the work needed to sketch a graph. If you can recognize that a function is even or odd, then you can often sketch a smaller portion of the graph and use symmetry to complete the picture.
    🔗
    Note that the idea of symmetry was first introduced in an algebraic context in the section on functions, but here we are using it as a tool for graphing.
    🔗

    Definition 3.7.4. Even and Odd Functions.

    • A function \(f\) is even if \(f(-x)=f(x)\) for all \(x\) in its domain. Visually, the graph of even functions are symmetric about the \(y\)-axis, a mirror image on either side.
      🔗
      🔗
    • A function \(f\) is odd if \(f(-x)=-f(x)\) for all \(x\) in its domain. Visually, the graph of odd functions has 180-degree (\(\pi\) radians) rotational symmetry about the origin.
      🔗
      🔗
    🔗

    Example 3.7.7.

    For each of the following functions, determine whether it is even, odd, both, or neither.
    🔗
    (a)
    \(f(x)=x^4-2x^2+1\)
    🔗
    Solution.
    Even. We have \(f(-x)=(-x)^4-2(-x)^2+1=x^4-2x^2+1=f(x).\)
    🔗
    🔗
    🔗
    (b)
    \(g(x)=x^3+5\sin(x)\)
    🔗
    Solution.
    Odd. We have \(g(-x)=(-x)^3+5\sin(-x)=-x^3-5\sin(x)=-(x^3+5\sin(x))=-g(x).\)
    🔗
    Note: that you need to remember from your favorite trig class that \(\sin(x)\) is an odd function, so \(\sin(-x)=-\sin(x)\text{.}\) It is also true that \(\cos(x)\) is an even function, so \(\cos(-x)=\cos(x)\text{.}\)
    🔗
    🔗
    🔗
    (c)
    \(h(x)=\dfrac{x^3+1}{x^2-1}\)
    🔗
    Solution.
    Neither. We have \(h(-x)=\frac{(-x)^3+1}{(-x)^2-1}=\frac{-x^3+1}{x^2-1}.\) This is not equal to \(h(x)\text{,}\) so it is not even. It is also not equal to \(-h(x)\text{,}\) so it is not odd.
    🔗
    We can also use a specific counterexample to show that \(h\) is not even or odd. For instance, \(h(2)=\dfrac{9}{3}=3\text{,}\) but \(h(-2)=\dfrac{-7}{3}\neq 3\) and \(h(-2)\neq -3\text{.}\)
    🔗
    🔗
    🔗
    🔗
    🔗

    Subsection Putting It All Together

    Now that we have developed all of the necessary tools, we are ready to apply the full curve sketching method to a variety of functions.
    🔗
    Below is a checklist of the types of steps that might be required in a curve sketching problem. Not every problem will require every step. You should focus only on the parts that are explicitly requested.
    🔗
    In particular, on timed quizzes and exams, it is important to balance completeness with efficiency. Problems are designed so that the requested steps can reasonably be completed within the allotted time.
    🔗

    Strategy 3.7.8. Curve Sketching Checklist.

    1. Compute \(f'(x)\) and \(f''(x)\text{.}\)
      🔗
    2. Find critical points from \(f'(x)=0\) and where \(f'(x)\) is undefined.
      🔗
    3. Use the sign of \(f'(x)\) to determine where the function increases or decreases and to classify critical points.
      🔗
    4. Find possible inflection points from \(f''(x)=0\) and where \(f''(x)\) is undefined.
      🔗
    5. Use the sign of \(f''(x)\) to determine concavity and confirm inflection points.
      🔗
    6. Find intercepts and identify any vertical asymptotes from the domain.
      🔗
    7. Describe end behavior as \(x\to\pm\infty\) using our horizontal asymptote techniques.
      🔗
    8. Check for even/odd symmetry, \(f(-x)=f(x)\) or \(f(-x)=-f(x)\text{.}\)
      🔗
    9. Combine these features into a clear sketch.
      🔗
    🔗

    Example 3.7.9.

    Consider the function \(s(x)=x-2\sin(x)\) on the interval \([-2\pi,2\pi]\text{.}\)
    🔗
    (a) Identify vertical and horizontal asymptotes, and symmetry if they exist.
    🔗
    (b) Find critical points and regions of increase and decrease.
    🔗
    (c) Find inflection points and intervals of concavity.
    🔗
    (d) Combine parts (a)-(c) to sketch the graph of \(s\text{.}\) Plot all critical points and inflection points.
    🔗
    Solution 1. Video Solution
    Here is a video solution to this problem.
    🔗
    🔗
    🔗
    Solution 2. Written Solution
    (a) There are no vertical asymptotes since the domain is all real numbers. There are no horizontal asymptotes since the function grows without bound as \(x\to\pm\infty\text{.}\)
    🔗
    The function is odd since,
    \begin{equation*} \begin{array}{rcl} s(-x) \amp =\amp -x-2\sin(-x)\\ \amp =\amp -x+2\sin(x)\\ \amp =\amp -(x-2\sin(x))=-s(x) \end{array} \end{equation*}
    so it has 180-degree rotational symmetry about the origin.
    🔗
    (b) The derivative is \(s'(x)=1-2\cos(x)\text{.}\) This is always defined, so there are no critical points from undefined derivatives. So we only need to solve \(s'(x)=0\) to find critical points.
    \begin{equation*} \begin{array}{rcl} 0 \amp =\amp 1-2\cos(x) \\ \cos(x)\amp = \amp \tfrac12 \\ x \amp= \amp\tfrac{-5\pi}{3}, \tfrac{-\pi}{3}, \tfrac{\pi}{3}, \tfrac{5\pi}{3} \end{array} \end{equation*}
    🔗
    Note that the symmetry of the function means that the critical points are symmetric about the origin, and the behavior on \([-2\pi,0]\) is a mirror image of the behavior on \([0,2\pi]\text{.}\)
    🔗
    Choosing test points in each interval determined by the critical points gives the following sign chart for \(s'(x)\text{:}\)
    🔗
    \begin{equation*} \begin{array}{c|c|c} x \amp \text{Sign of } s'(x) \amp \text{Behavior of } s(x) \\ \hline -2\pi \amp (-) \amp \text{decreasing} \\ -\tfrac{5\pi}{3} \amp 0 \amp \\ -\pi \amp (+) \amp \text{increasing} \\ -\tfrac{\pi}{3} \amp 0 \amp \\ 0 \amp (-) \amp \text{decreasing} \\ \tfrac{\pi}{3} \amp 0 \amp \\ \pi \amp (+) \amp \text{increasing} \\ \tfrac{5\pi}{3} \amp 0 \amp \\ 2\pi \amp (-) \amp \text{decreasing} \end{array} \end{equation*}
    So the function is decreasing on \([-2\pi,-\tfrac{5\pi}{3}) \cup (-\tfrac{\pi}{3},\tfrac{\pi}{3}) \cup (\tfrac{5\pi}{3},2\pi]\text{,}\) and increasing on \((-\tfrac{5\pi}{3},-\tfrac{\pi}{3}) \cup (\tfrac{\pi}{3},\tfrac{5\pi}{3})\text{.}\)
    🔗
    Again we could have used the symmetry of the function to reduce our work. For instance, we could have just analyzed the behavior on \([0,2\pi]\) and then used symmetry to get the rest of our conclusion.
    🔗
    (c) The second derivative is \(s''(x)=2\sin(x)\text{,}\) which is defined everywhere, so there are no inflection points from undefined derivatives. To find inflection points, we solve \(s''(x)=0\text{,}\) which gives \(\sin(x)=0\text{.}\) This happens at \(x= -2\pi, -\pi, 0, \pi, 2\pi\text{.}\) Because \(x=-2\pi\) and \(x=2\pi\) are endpoints of the interval, they cannot be inflection points, so the only possible inflection points are at \(x=-\pi,0,\pi\text{.}\)
    🔗
    To determine intervals of concavity, we can use test points in each interval determined by the possible inflection points. This gives the following sign chart for \(s''(x)\text{:}\)
    🔗
    \begin{equation*} \begin{array}{c|c|c} x \amp \text{Sign of } s''(x) \amp \text{Concavity of } s(x) \\ \hline -2\pi \amp 0 \amp \\ -\tfrac{3\pi}{2} \amp (+) \amp \text{concave up} \\ -\pi \amp 0 \amp \\ -\tfrac{\pi}{2} \amp (-) \amp \text{concave down} \\ 0 \amp 0 \amp \\ \tfrac{\pi}{2} \amp (+) \amp \text{concave up} \\ \pi \amp 0 \amp \\ \tfrac{3\pi}{2} \amp (-) \amp \text{concave down} \\ 2\pi \amp 0 \amp \end{array} \end{equation*}
    So the function is concave up on \((-2\pi,-\pi) \cup (0,\pi)\) and concave down on \((-\pi,0) \cup (\pi,2\pi)\text{.}\)
    🔗
    Because the function changes concavity at \(x=-\pi,0,\pi\text{,}\) these points are indeed inflection points.
    🔗
    (d) Combining all of this information, we can sketch the graph of \(s\text{.}\) We know there are no asymptotes, and the function is odd, so we can focus on sketching the portion of the graph on \([0,2\pi]\) and then use symmetry to get the rest of the graph.
    🔗
    Since we will need to plot all critical points and inflection points, we should compute the function values at these points.
    🔗
    \begin{equation*} \begin{array}{c|c|c} x \amp s(x) \amp \text{Type of Point} \\ \hline 0 \amp 0 \amp \text{inflection point} \\ \tfrac{\pi}{3} \amp \tfrac{\pi}{3}-\sqrt{3}\approx -0.7 \amp \text{critical point} \\ \pi \amp \pi \amp \text{inflection point} \\ \tfrac{5\pi}{3} \amp \tfrac{5\pi}{3}+\sqrt{3}\approx 6.7 \amp \text{critical point} \\ \end{array} \end{equation*}
    Plotting these points and using the behavior determined in parts (b) and (c) gives the following sketch of the graph of \(s\text{:}\)
    🔗
    Remark 3.7.11.
    Note that this problem was designed to be completed in a reasonable amount of time on a quiz or exam, so it is not meant to be an exhaustive analysis of the function. For example it did not ask for intercepts, so we did not compute them. It also did not ask us to classify critical points as local maxima or minima, so we just identified them as critical points and used the sign of the derivative to determine where the function increases or decreases.

    🔗

    Example 3.7.12.

    Consider the function \(f(x) = \dfrac{1}{1+e^{-x}}\) with derivatives
    \begin{equation*} f'(x)=\dfrac{e^{-x}}{(1+e^{-x})^2} \quad \text{ and } \quad f''(x)=\dfrac{e^{-x}(e^{-x}-1)}{(1+e^{-x})^3} \end{equation*}

    (a) Identify vertical and horizontal asymptotes, intercepts and symmetry.
    🔗
    (b) Find critical points and regions of increase and decrease. Classify critical points as local maxima, local minima, or neither.
    🔗
    (c) Find intervals of concavity and inflection points.
    🔗
    (d) Combine parts (a)-(c) to sketch the graph of \(f\text{.}\)

    Solution 2. Written Answers
    Here are the answers to this problem to help you check your work. More detailed explanations are available in the video solution.
    🔗

    🔗
    (b) The derivative is \(f'(x)=\dfrac{e^{-x}}{(1+e^{-x})^2}\text{,}\) which is defined everywhere, so there are no critical points from undefined derivatives. Setting the derivative equal to zero gives \(e^{-x}=0\text{,}\) which has no solutions, so there are no critical points from \(f'(x)=0\) either. So there are no critical points. Choosing any test point in \((-\infty,\infty)\text{,}\) we see that \(f'(x)>0\) for all \(x\text{,}\) so the function is increasing everywhere.

Setting the second derivative equal to zero gives \(e^{-x}(e^{-x}-1)=0\text{,}\) which has a solution at \(x=0\text{.}\) Choosing test points in \((-\infty,0)\) and \((0,\infty)\) shows that the function is concave up on \((-\infty,0)\) and concave down on \((0,\infty)\text{,}\) so there is an inflection point at \(x=0\text{.}\)

    Some True/False Problems.

    Determine whether each statement is true or false and briefly justify your reasoning.
    🔗
    1.
    If \(f'(1)=0\) and \(f''(1)>0\text{,}\) then \(f\) has a local minimum at \(x=1\text{.}\)
    🔗
    Solution.
    True. This is exactly the conclusion of the Second Derivative Test.
    🔗
    🔗
    🔗
    2.
    If \(f'(5)=0\) and \(f''(5)=0\text{,}\) then \(f\) has neither a local minimum nor a local maximum at \(x=5\text{.}\)
    🔗
    Solution.
    False. If \(f''(5)=0\text{,}\) the Second Derivative Test is inconclusive, so we cannot conclude anything about local minima or maxima at \(x=5\text{.}\) For instance, the function \(f(x)=(x-5)^4\) has \(f'(5)=0\) and \(f''(5)=0\text{,}\) but it has a local minimum at \(x=5\text{.}\)
    🔗
    🔗
    🔗
    3.
    If \(f''(c)<0\text{,}\) then \(f\) must be decreasing at \(x=c\text{.}\)
    🔗
    Solution.
    False. If \(f''(c)<0\text{,}\) then the function is concave down at \(x=c\text{,}\) but it could be increasing, decreasing, or constant at that point. For instance, the function \(f(x)=-x^2\) has \(f''(0)=-2<0\text{,}\) but it is increasing at \(x=-1\text{.}\)
    🔗
    🔗
    🔗
    4.
    If \(f\) is an odd function with a local maximum at \(x=2\text{,}\) then there is a local minimum at \(x=-2\text{.}\)
    🔗
    Solution.
    True. If \(f\) is odd, then \(f(-2)=-f(2)\text{.}\) So if there is a local maximum at \(x=2\text{,}\) then there is a local minimum at \(x=-2\text{.}\)
    🔗
    🔗
    🔗
    5.
    If \(f\) is an even function with a local maximum at \(x=2\text{,}\) then there is a local minimum at \(x=-2\text{.}\)
    🔗
    Solution.
    False. If \(f\) is even, then \(f(-2)=f(2)\text{.}\) So if there is a local maximum at \(x=2\text{,}\) then there is also a local maximum at \(x=-2\text{.}\)
    🔗
    🔗
    🔗
    6.
    If \(f\) has symmetry (either even or odd) and has an inflection point at \(x=3\text{,}\) then there is also an inflection point at \(x=-3\text{.}\)
    🔗
    Solution.
    True. If \(f\) is even, then \(f''(-3)=f''(3)=0\) and the concavity changes at \(x=3\text{,}\) so there is an inflection point at \(x=-3\text{.}\) If \(f\) is odd, then \(f''(-3)=-f''(3)=0\) and the concavity changes at \(x=3\text{,}\) so there is also an inflection point at \(x=-3\text{.}\)
    🔗
    🔗
    🔗
    7.
    If \(f\) is even, then \(f'(x)\) is odd (whenever \(f'\) exists).
    🔗
    Solution.
    True. If \(f\) is even, then \(f(-x)=f(x)\text{.}\) Differentiating both sides gives \(-f'(-x)=f'(x)\text{,}\) so \(f'(-x)=-f'(x)\text{,}\) which means that \(f'\) is odd.
    🔗
    🔗
    🔗
    8.
    The function \(p(x)=x^5-2x^3+1\) is odd.
    🔗
    Solution.
    False. We have
    \begin{equation*} p(-x)=(-x)^5-2(-x)^3+1=-x^5+2x^3+1\neq -(x^5-2x^3+1)=-p(x). \end{equation*}
    🔗
    🔗
    🔗
    9.
    The function \(q(x)=\dfrac{\sin(x)}{x^3+x}\) is even.
    🔗
    Solution.
    True. We have
    \begin{equation*} q(-x)=\frac{\sin(-x)}{(-x)^3+(-x)}=\frac{-\sin(x)}{-x^3-x}=\frac{\sin(x)}{x^3+x}=q(x), \end{equation*}
    so the function is even.
    🔗
    🔗
    🔗
    10.
    The function \(f(x)=\dfrac{\ln x}{x} \) is decreasing on \((4,6)\text{.}\)
    🔗
    Solution.
    True. The derivative is \(f'(x)=\dfrac{1-\ln x}{x^2}\text{.}\) For \(x\in(4,6)\text{,}\) we have \(\ln x>1\text{,}\) so \(f'(x)<0\text{,}\) so the function is decreasing on \((4,6)\text{.}\)
    🔗
    🔗
    🔗
    11.
    The function \(f(x)=\dfrac{\ln x}{x} \) has exactly one critical point.
    🔗
    Solution.
    True. The derivative is \(f'(x)=\dfrac{1-\ln x}{x^2}\text{,}\) which is defined everywhere on the domain of \(f\text{,}\) so there are no critical points from undefined derivatives. Setting the derivative equal to zero gives \(1-\ln x=0\text{,}\) which has a solution at \(x=e\text{.}\) So there is exactly one critical point from \(f'(x)=0\text{,}\) and there are no critical points from undefined derivatives, so there is exactly one critical point in total.
    🔗
    🔗
    🔗
    12.
    The function \(f(x)=\dfrac{\ln x}{x} \) has exactly one inflection point.
    🔗
    Solution.
    True. The second derivative is \(f''(x)=\dfrac{2\ln x-3}{x^3}\text{,}\) which is defined everywhere on the domain of \(f\text{,}\) so there are no inflection points from undefined derivatives. Setting the second derivative equal to zero gives \(2\ln x-3=0\text{,}\) which has a solution at \(x=e^{3/2}\text{.}\) We can check by testing the sign of \(f''(x)\) on either side of \(x=e^{3/2}\) that it does change signs. For example, \(f''(1)=\dfrac{2\ln 1-3}{1^3}=-3<0\) and \(f''(e^2)=\dfrac{2\ln e^2-3}{(e^2)^3}=\dfrac{4-3}{e^6}=\dfrac{1}{e^6}>0\text{.}\) So there is exactly one inflection point from \(f''(x)=0\text{,}\) and there are no inflection points from undefined derivatives.
    🔗
