    Section 3.4 Extrema

    Objectives

    • Define and distinguish between absolute and local maxima and minima.
      🔗
    • Use the Extreme Value Theorem to determine when absolute extrema are guaranteed to exist.
      🔗
    • Identify critical points of a function.
      🔗
    • Apply derivatives to locate and classify extrema on a closed interval.
      🔗
    • Understand how discontinuities and nondifferentiable points affect extrema.
      🔗
    🔗
    In many real-world problems, we are not just interested in how a function behaves, but in finding its largest or smallest possible values. For example, we might want to know the maximum height of a projectile, the minimum cost of producing an item, or the highest temperature reached during a day. Problems like these lead us to the study of extrema.
    🔗

    Subsection Absolute Maxima and Minima

    Definition 3.4.1. Absolute Maximum and Minimum.

    Let \(f\) be a function defined on a closed interval \([a,b]\text{.}\)
    🔗
    An absolute maximum of \(f\) on \([a,b]\) occurs at a point \(x=c\) in \([a,b]\) if \(f(c)\ge f(x)\) for all \(x\in[a,b]\text{.}\)
    🔗
    An absolute minimum of \(f\) on \([a,b]\) occurs at a point \(x=c\) in \([a,b]\) if \(f(c)\le f(x)\) for all \(x\in[a,b]\text{.}\)
    🔗
    🔗
    The output values \(f(c)\) are called the maximum value and minimum value of the function. While these values are unique, it is possible for a function to attain the same maximum or minimum value at more than one input.
    🔗

    Remark 3.4.2.

    The terms maximum, minimum, and extremum come from Latin. Their plural forms are maxima, minima, and extrema.
    🔗
    🔗

    Theorem 3.4.3. Extreme Value Theorem.

    If a function \(f\) is continuous on a closed, finite interval \([a,b]\text{,}\) then \(f\) has at least one absolute maximum and at least one absolute minimum on \([a,b]\text{.}\)
    🔗
    🔗

    Remark 3.4.4.

    This theorem does not tell us where the extrema occur, only that they must exist. If the function is not continuous or the interval is not closed, absolute extrema may fail to exist.
    🔗
    🔗
    🔗

    Subsection Local Maxima and Minima

    Absolute extrema describe the highest and lowest points over an entire interval. A broader concept focuses only on nearby behavior.
    🔗

    Definition 3.4.5. Local Maximum and Minimum.

    A local maximum of a function \(f\) occurs at \(x=c\) if \(f(c)\ge f(x)\) for all \(x\) sufficiently close to \(c\text{.}\)
    🔗
    A local minimum of a function \(f\) occurs at \(x=c\) if \(f(c)\le f(x)\) for all \(x\) sufficiently close to \(c\text{.}\)
    🔗
    🔗
    Intuitively, local maxima correspond to hilltops and local minima correspond to valley bottoms. Every absolute maximum or minimum is also a local extremum, but not every local extremum is an absolute one.
    🔗
    🔗

    Subsection Derivatives and Extrema

    Calculus provides a powerful tool for locating extrema. If a function has a smooth hilltop or valley bottom at an interior point, the tangent line there is horizontal.
    🔗

    Theorem 3.4.6. First Derivative Test for Extrema.

    If \(f\) has a local maximum or minimum at an interior point \(x=c\) and \(f\) is differentiable at \(c\text{,}\) then \(f'(c)=0\text{.}\)
    🔗
    🔗

    Remark 3.4.7.

    The converse is not true. A point where \(f'(c)=0\) need not be a maximum or minimum.
    🔗
    🔗
    🔗

    Subsection Critical Points

    Definition 3.4.8. Critical Point.

    A critical point (or critical number) of a function \(f\) is a point \(x=c\) where either
    🔗
    \begin{equation*} f'(c)=0 \quad\text{or}\quad f'(c)\text{ is undefined}. \end{equation*}
    🔗
    Critical points are the only possible locations of local extrema in the interior of an interval. Endpoints of the interval must also be considered when searching for absolute extrema.
    🔗
    🔗

    Subsection A Method for Finding Absolute Extrema

    Strategy 3.4.9. Finding Absolute Maxima and Minima.

    To find the absolute maximum and minimum of a function \(f\) on a closed interval \([a,b]\text{:}\)
    🔗
    1. Find all critical points of \(f\) in \([a,b]\text{.}\)
      🔗
    2. Evaluate \(f\) at each critical point and at the endpoints \(a\) and \(b\text{.}\)
      🔗
    3. The largest value is the absolute maximum; the smallest value is the absolute minimum.
      🔗
    4. If \(f\) is not continuous, also examine behavior near discontinuities.
      🔗
    🔗
    🔗

    Subsection Examples and Common Pitfalls

    Example 3.4.10.

    Consider \(f(x)=x^3\text{.}\)
    🔗
    Solution.
    We compute \(f'(x)=3x^2\text{,}\) so the only critical point is \(x=0\text{.}\) However, the graph has no local maximum or minimum at this point. Instead, \(x=0\) is a stationary point where the function briefly levels off before continuing to increase.
    🔗
    🔗
    🔗

    Example 3.4.11.

    Let \(f(x)=|2x^2+2x-1|\text{.}\)
    🔗
    Solution.
    The derivative is undefined when the expression inside the absolute value is zero, producing corners in the graph. These points must be included as critical points, and in fact they correspond to the absolute minimum values of the function.
    🔗
    🔗
    🔗

    Example 3.4.12.

    Let \(f(x)=x^2+\dfrac{1}{(x-1)^2}\) on \([-2,2]\text{.}\)
    🔗
    Solution.
    Although \(f\) has critical points and endpoint values, the function increases without bound near \(x=1\text{.}\) Therefore, \(f\) has no absolute maximum on this interval, but it does have an absolute minimum.
    🔗
    🔗
    🔗
    🔗

    Exercises Practice / Study Problems

    Conceptual Questions.

    1.
    True or False: If \(f'(c)=0\text{,}\) then \(x=c\) is a local maximum or minimum.
    🔗
    Solution.
    False. A zero derivative may correspond to a stationary point.
    🔗
    🔗
    🔗
    2.
    True or False: A continuous function on \([a,b]\) must have an absolute maximum.
    🔗
    Solution.
    True. This follows from the Extreme Value Theorem.
