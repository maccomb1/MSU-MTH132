Example 1.2.8. Using a Conjugate.

Evaluate \(\displaystyle \lim_{x\to 9} \frac{x-9}{\sqrt{x}-3}\text{.}\)
ðŸ”—
Solution.
Direct substitution gives the meaningless expression \(\tfrac{0}{0}\text{,}\) so we must simplify the expression first.
ðŸ”—
Multiply numerator and denominator by the conjugate \(\sqrt{x}+3\text{:}\)
ðŸ”—
\begin{equation*} \displaystyle\lim_{x\to 9}\frac{x-9}{\sqrt{x}-3}\cdot\frac{\sqrt{x}+3}{\sqrt{x}+3} = \lim_{x\to 9}\frac{(x-9)(\sqrt{x}+3)}{(\sqrt{x})^2-3^2} \end{equation*}
Simplify the denominator using the difference of squares:
ðŸ”—
\begin{equation*} \displaystyle\lim_{x\to 9}\frac{(x-9)(\sqrt{x}+3)}{x-9} \end{equation*}
Now cancel the common factor \(x-9\text{:}\)
ðŸ”—
\begin{equation*} \displaystyle\lim_{x\to 9} (\sqrt{x}+3) = 6. \end{equation*}
