    Subsection 1.3.1 Definition and Overview

    One of the most basic features of a function is whether it is continuous. Roughly, this means that a small change in \(x\) always leads to a fairly small change in \(f(x)\text{,}\) without instantaneous jumps. In physical terms, the position of a particle moving in space is continuous, but the position displayed in a video could have a gap, making the position function jump discontinuously.
    ğŸ”—

    Definition 1.3.1.

    A function \(f(x)\) is continuous at \(x=a\) whenever
    \begin{equation*} \displaystyle\lim_{x\to a} f(x) = f(a). \end{equation*}
    ğŸ”—
    ğŸ”—
    Graphically, a function is continuous if the curve \(y=f(x)\) passes through the point \((a,f(a))\) without any jumps or holes. This matches the intuitive idea of continuity â€” you can draw the graph near \(x=a\) without lifting your pencil.
    ğŸ”—
    We say \(f(x)\) is continuous on an interval whenever it is continuous at each point of the interval. For endpoints of a closed interval \([a,b]\text{,}\) we require one-sided limits: \(\displaystyle\lim_{x\to a^+} f(x) = f(a)\) and \(\displaystyle\lim_{x\to b^-} f(x) = f(b)\text{.}\)
    ğŸ”—
    ğŸ”—

    Subsection 1.3.2 Removable Discontinuity

    Definition 1.3.2.

    A function has a removable discontinuity at \(x=a\) if \(\displaystyle\lim_{x\to a} f(x)\) exists but \(f(a)\) is either undefined or does not equal that limit.
    ğŸ”—
    Graphically, a removable discontinuity appears as a hole in the curve \(y=f(x)\) at \(x=a\text{.}\) The discontinuity is "removable" because we can redefine \(f(a)\) to equal the limit, making the function continuous at \(x=a\text{.}\)
    ğŸ”—
    ğŸ”—

    Example 1.3.4.

    Revisit ExampleÂ 1.2.5. Verify that the function \(f(x)=\dfrac{x^2-4x+4}{x^2-x-2}\) has a removable discontinuity at \(x=2\text{.}\)
    ğŸ”—
    Solution.
    At \(x=2\text{,}\) both the numerator and denominator vanish, giving us the \(\tfrac00\) meaningless expression so the function is undefined. However, we can verify again that \(\displaystyle\lim_{x\to 2} f(x) \) exists (is equal to \(0\)). This matches the definition of a removable discontinuity. Moreover we can view the picture in ExampleÂ 1.2.5 to see visually the hole in the graph at \(x=2\text{.}\)
    ğŸ”—
    ğŸ”—
    ğŸ”—

    Example 1.3.5.

    Revisit ExampleÂ 1.2.8. Verify that this function \(f(x)=\dfrac{x-9}{\sqrt{x}-3}\) has a removable discontinuity at \(x=9\text{.}\)
    ğŸ”—
    Solution.
    At \(x=9\text{,}\) both the numerator and denominator vanish, giving the undefined form \(\tfrac{0}{0}\text{.}\) However, multiplying numerator and denominator by the conjugate simplifies the expression to \(\sqrt{x}+3\text{,}\) which is continuous everywhere. Taking the limit, we find
    \begin{equation*} \lim_{x\to 9}\frac{x-9}{\sqrt{x}-3} = \lim_{x\to 9} (\sqrt{x}+3) = 6. \end{equation*}
    Thus, although \(f(9)\) is undefined, the limit exists and equals 6. This is exactly the definition of a removable discontinuity, and the graph shows a â€œholeâ€ at \(x=9\text{.}\)

    Subsection 1.3.3 Jump Discontinuity

    Definition 1.3.6.

    A function has a jump discontinuity at \(x=a\) if the left and right limits exist but are unequal:
    \begin{equation*} \displaystyle\lim_{x\to a^-} f(x) \neq \lim_{x\to a^+} f(x). \end{equation*}
    ğŸ”—
    Graphically, a jump discontinuity appears as a sudden jump in the curve \(y=f(x)\) at \(x=a\text{.}\)
    ğŸ”—
    ğŸ”—

    Example 1.3.8.

    Revisit ExampleÂ 1.2.10. Verify that the function \(f(x)=\dfrac{|x^2-4|}{x-2}\) has a jump discontinuity at \(x=2\text{.}\)
    ğŸ”—
    Solution.
    Notice that \(x^2-4=(x-2)(x+2)\text{.}\) For \(x>2\text{,}\) the absolute value simplifies to \(|x^2-4|=(x-2)(x+2)\text{,}\) so \(f(x)=\dfrac{(x-2)(x+2)}{x-2}=x+2\text{.}\) For \(x<2\text{,}\) we instead get \(|x^2-4|=-(x-2)(x+2)\text{,}\) so \(f(x)=\dfrac{-(x-2)(x+2)}{x-2}=-(x+2)\text{.}\)
    ğŸ”—
    Thus the left-hand limit is
    \begin{equation*} \lim_{x\to 2^-} f(x) = \lim_{x\to 2^-} (-(x+2)) = -4, \end{equation*}
    while the right-hand limit is
    \begin{equation*} \lim_{x\to 2^+} f(x) = \lim_{x\to 2^+} (x+2) = 4. \end{equation*}
    Since these limits are not equal, the two-sided limit does not exist. This confirms that \(f\) has a jump discontinuity at \(x=2\text{.}\)
    ğŸ”—
    ğŸ”—
    ğŸ”—
    ğŸ”—

    Subsection 1.3.4 Infinite Discontinuity

    Definition 1.3.9.

    A function has an infinite discontinuity at \(x=a\) if one or both one-sided limits diverge to infinity:
    \begin{equation*} \lim_{x\to a^\pm} f(x) = \pm\infty. \end{equation*}
    ğŸ”—
    Graphically, an infinite discontinuity appears as a vertical asymptote in the curve \(y=f(x)\) at \(x=a\text{.}\)
    ğŸ”—
    ğŸ”—

    Example 1.3.11.

    Revisit ExampleÂ 1.1.15. Verify that the function \(f(x)=\dfrac{1}{|x|}\) has an infinite discontinuity at \(x=0\text{.}\)
    ğŸ”—
    Solution.
    The function \(f(x)=\tfrac{1}{|x|}\) is undefined at \(x=0\text{.}\) As \(x\to 0^+\text{,}\) \(|x|=x\text{,}\) so \(f(x)=\tfrac{1}{x}\to +\infty\text{.}\) As \(x\to 0^âˆ’\text{,}\) \(|x|=-x\text{,}\) so \(f(x)=\tfrac{1}{-x}=\tfrac{1}{|x|}\to +\infty\text{.}\) In both directions the function grows beyond all bounds, so
    \begin{equation*} \lim_{x\to 0} \tfrac{1}{|x|}=+\infty. \end{equation*}
    ğŸ”—
    Thus, \(f(x)\) has an infinite discontinuity at \(x=0\text{.}\) Graphically, this appears as a vertical asymptote along the \(y\)-axis.
    ğŸ”—
    ğŸ”—
    ğŸ”—
    ğŸ”—

    Subsection 1.3.5 Essential Discontinuity

    Definition 1.3.12.

    A function has an essential discontinuity at \(x=a\) if one or both one-sided limits fail to exist in any sense.
    ğŸ”—
    Graphically, an essential discontinuity quite often appears as a wild oscillation in the curve \(y=f(x)\) at \(x=a\text{.}\)
    ğŸ”—
    ğŸ”—

    Example 1.3.14.

    Consider the function \(f(x)=\sin\!\left(\tfrac{1}{x}\right)\text{.}\) Show that \(f(x)\) has an essential discontinuity at \(x=0\text{.}\)
    ğŸ”—
    Solution.
    The function is not defined at \(x=0\text{.}\) As \(x\) approaches 0, the reciprocal \(\tfrac{1}{x}\) grows without bound in magnitude. Since \(\sin(\theta)\) oscillates between \(-1\) and \(1\) for all values of \(\theta\text{,}\) the outputs \(\sin\!\left(\tfrac{1}{x}\right)\) continue to oscillate rapidly between \(-1\) and \(1\) as \(x\to 0\text{.}\)
    ğŸ”—
    No single limiting value can be assigned: the left-hand and right-hand limits both fail to exist because of these wild oscillations. By definition, this is an essential discontinuity at \(x=0\text{.}\)
    ğŸ”—
    Graphically, the curve appears to â€œwiggle infinitely fastâ€ near the vertical line \(x=0\text{,}\) matching the intuition for an essential discontinuity.
    ğŸ”—
    ğŸ”—

    Subsection 1.3.6 Continuity of Piecewise Functions

    For a function defined by cases, whose graph consists of several continuous pieces, the function is continuous provided the pieces join together at the transition points where the cases meet.
    ğŸ”—

    Example 1.3.15.

    Consider the piecewise function
    ğŸ”—
    \begin{equation*} f(x) = \begin{cases} x^2 - 1 & \text{for } x < 2, \\[6pt] kx - 3 & \text{for } x \geq 2. \end{cases} \end{equation*}
    Find the value of \(k\) that makes \(f(x)\) continuous at \(x=2\text{.}\)
    ğŸ”—
    Solution.
    To check continuity at \(x=2\text{,}\) we need three things: (1) the function is defined at \(x=2\text{,}\) (2) the limit \(\displaystyle\lim_{x\to 2} f(x)\) exists, and (3) the limit equals \(f(2)\text{.}\)
    ğŸ”—
    First, \(f(2)=k(2)-3=2k-3\text{,}\) so the function is defined at \(x=2\) for any value of \(k\text{.}\)
    ğŸ”—
    Next, compute the one-sided limits. For \(x\to 2^âˆ’\text{,}\) we use the left-hand piece \(x^2-1\text{:}\)
    \begin{equation*} \lim_{x\to 2^-} f(x) = \lim_{x\to 2^-} (x^2-1) = 2^2-1=3. \end{equation*}
    For \(x\to 2^+\text{,}\) we use the right-hand piece \(kx-3\text{:}\)
    \begin{equation*} \lim_{x\to 2^+} f(x) = \lim_{x\to 2^+} (kx-3) = 2k-3. \end{equation*}
    ğŸ”—
    For the limit to exist, these must be equal:
    \begin{equation*} 3 = 2k - 3. \end{equation*}
    Solving gives \(2k=6\text{,}\) so \(k=3\text{.}\)
    ğŸ”—
    Finally, with \(k=3\) we have \(f(2)=3\text{,}\) which matches the limit. Therefore \(f(x)\) is continuous at \(x=2\) when \(k=3\text{.}\)
    ğŸ”—
    ğŸ”—
    ğŸ”—

    Example 1.3.16.

    Suppose a weight is winched from the ground at a constant speed for 8 seconds, is dropped, and lands 2 seconds later.
    ğŸ”—
    Then the height \(s(t)\) in feet at time \(t\) seconds is given by
    ğŸ”—
    \begin{equation*} s(t) = \begin{cases} vt & \text{for } t \in [0,8), \\[6pt] 64 - 16(t-8)^2 & \text{for } t \in [8,10]. \end{cases} \end{equation*}
    How fast should the winch haul upward?
    ğŸ”—
    Solution.
    The function \(s(t)\) is defined in two pieces. The first piece, \(s(t)=vt\text{,}\) describes the weight being lifted upward from the ground at a constant speed \(v\) feet per second, for times \(0 \leq t < 8\text{.}\)
    ğŸ”—
    The second piece, \(s(t)=64-16(t-8)^2\text{,}\) describes what happens once the weight is released at \(t=8\text{.}\) Recall that a dropped object falls \(16t^2\) feet in \(t\) seconds. Since the fall begins at \(t=8\text{,}\) we measure the time of fall as \((t-8)\text{.}\) Thus, after \((t-8)\) seconds of falling, the weight will have fallen \(16(t-8)^2\) feet. The weight was released from a height of 64 feet, so its position after \(t=8\) seconds is given by \(64 - 16(t-8)^2\text{.}\)
    ğŸ”—
    To find the correct speed \(v\) of the winch, we need the two pieces of the function to â€œmeet upâ€ at \(t=8\text{.}\) That means the height from the lifting phase at \(t=8\) must equal the height from the falling phase at \(t=8\text{.}\) Using the lifting phase we get \(s(8)=v\cdot 8\text{.}\) Using the falling phase we get \(s(8)=64-16(8-8)^2=64\text{.}\) Setting these equal gives
    \begin{equation*} v \cdot 8 = 64 \quad \Longrightarrow \quad v = \tfrac{64}{8} = 8 \text{ ft/sec}. \end{equation*}
    ğŸ”—
    Finally, let us check continuity explicitly at \(t=8\text{.}\) From the left-hand side,
    \begin{equation*} \lim_{t \to 8^-} s(t) = \lim_{t \to 8^-} vt = 8 \cdot 8 = 64. \end{equation*}
    From the right-hand side,
    \begin{equation*} \lim_{t \to 8^+} s(t) = \lim_{t \to 8^+} \big(64 - 16(t-8)^2\big) = 64. \end{equation*}
    Since both one-sided limits equal 64, the two-sided limit exists and equals 64. Moreover, this matches the value of the function at that point, \(s(8)=64\text{.}\) Therefore the function is continuous at \(t=8\text{.}\)
    ğŸ”—
    ğŸ”—
    ğŸ”—

    Subsection 1.3.7 Domain of Continuity

    Almost all functions defined by formulas are continuous, except at points where they are undefined. This follows from our methods for computing limits.
    ğŸ”—

    Example 1.3.17.

    Find where \(g(x)=\dfrac{(x^2-3x+1)\sqrt{x+1}}{x-3}\) is continuous.
    ğŸ”—
    Solution.
    First, letâ€™s analyze the rational factor outside the square root. Using the Limit Laws from SectionÂ 1.2 repeatedly, we compute:
    ğŸ”—
    \begin{equation*} \lim_{x\to a} \frac{x^2-3x+1}{x-3} = \dfrac{a^2 - 3a + 1}{a - 3}, \end{equation*}
    as long as the denominator does not vanish. In other words, this step is valid provided \(a \neq 3\text{.}\)
    ğŸ”—
    Next, consider the square root factor. By the Limit Laws,
    ğŸ”—
    \begin{equation*} \lim_{x\to a} \sqrt{x+1} = \sqrt{a+1}, \end{equation*}
    but this only makes sense when the inside of the square root is nonnegative. Thus we require \(a+1 \geq 0\text{,}\) or equivalently \(a > -1\) (strict inequality if we want the square root defined for all nearby \(x\)).
    ğŸ”—
    We now combine the two factors to evaluate the overall limit:
    ğŸ”—
    \begin{equation*} \lim_{x\to a} \dfrac{(x^2 - 3x + 1)\sqrt{x+1}}{x-3}. \end{equation*}
    By the Limit Laws, the limit of a product equals the product of the limits, provided each individual limit exists. So we can separate this into two pieces:
    ğŸ”—
    \begin{equation*} \left(\lim_{x\to a} \dfrac{x^2-3x+1}{x-3}\right)\cdot \left(\lim_{x\to a} \sqrt{x+1}\right). \end{equation*}
    We already computed each factor individually: the rational expression simplifies to \(\dfrac{a^2 - 3a + 1}{a - 3}\) (valid when \(a \neq 3\)), and the square root tends to \(\sqrt{a+1}\) (valid when \(a > -1\)).
    ğŸ”—
    \begin{equation*} \dfrac{(a^2 - 3a + 1)\sqrt{a+1}}{a-3}. \end{equation*}
    Thus, whenever both conditions (\(a \neq 3\) and \(a > -1\)) are satisfied, this is the value of the limit.
    ğŸ”—
    Letâ€™s now examine the exceptional cases:
    ğŸ”—
    • \(a < -1\text{:}\) The square root \(\sqrt{x+1}\) is undefined, so the function is not continuous.
      ğŸ”—
      ğŸ”—
    • \(a = -1\text{:}\) The square root is defined, and since this is the left endpoint of the domain, continuity only requires the right-hand limit. Indeed, \(\lim_{x \to -1^+} g(x) = g(-1)\text{,}\) so the function is continuous at \(a=-1\text{.}\)
      ğŸ”—
      ğŸ”—
    • \(a = 3\text{:}\) The denominator vanishes, and the graph has a vertical asymptote. This corresponds to an infinite discontinuity (type (iv)).
      ğŸ”—
      ğŸ”—
    Putting this all together, we conclude that \(g(x)\) is continuous everywhere in its domain, which consists of the intervals
    ğŸ”—
    \begin{equation*} [-1, 3) \cup (3, \infty). \end{equation*}

    Subsection 1.3.8 Compositions of Continuous Functions

    Theorem 1.3.19. Composition Law.

    If \(\displaystyle\lim_{x\to a} g(x)=b\) and \(f\) is continuous at \(x=b\text{,}\) then
    \begin{equation*} \lim_{x\to a} f(g(x)) = f(\lim_{x\to a} g(x)). \end{equation*}
    ğŸ”—
    ğŸ”—

    Proof.

    A full \(\epsilon\)â€“\(\delta\) proof is omitted here; it uses techniques developed in a proofâ€‘based course such as MTH320.
    ğŸ”—
    ğŸ”—

    Example 1.3.20.

    Evaluate
    \begin{equation*} \displaystyle \lim_{x\to 1} \sin\!\left(\frac{x^2-1}{x-1}\right). \end{equation*}
    ğŸ”—
    Solution.
    First, observe that the inside function is \(g(x)=\dfrac{x^2-1}{x-1}\text{.}\) Direct substitution gives \(\tfrac{0}{0}\text{,}\) so we simplify:
    ğŸ”—
    \begin{equation*} \frac{x^2-1}{x-1} = \frac{(x-1)(x+1)}{x-1} = x+1, \quad x\neq 1. \end{equation*}
    Thus, as \(x \to 1\text{,}\)
    \begin{equation*} \lim_{x\to 1} g(x) = \lim_{x\to 1} (x+1) = 2. \end{equation*}
    ğŸ”—
    Since the outer function is \(f(u)=\sin(u)\text{,}\) which is continuous everywhere, we may apply the Composition Law:
    ğŸ”—
    \begin{equation*} \lim_{x\to 1} \sin\!\left(\frac{x^2-1}{x-1}\right) = \sin\!\Big(\lim_{x\to 1} \frac{x^2-1}{x-1}\Big) = \sin(2). \end{equation*}
    Therefore, the limit is \(\sin(2)\text{.}\)

    1.
    If \(f\) is continuous at \(x=2\text{,}\) then \(\displaystyle\lim_{x\to 2} f(x)=f(2)\text{.}\)
    ğŸ”—
    Solution.
    True. This is exactly the definition of continuity at a point.
    ğŸ”—
    ğŸ”—
    ğŸ”—
    2.
    The function \(f(x)=\dfrac{x^2-4}{x-2}\) has a removable discontinuity at \(x=2\text{.}\)
    ğŸ”—
    Solution.
    True. At \(x=2\text{,}\) numerator and denominator both vanish, giving \(\tfrac{0}{0}\text{.}\) Simplifying, \(f(x)=x+2\) for \(x\neq 2\text{,}\) so \(\displaystyle\lim_{x\to 2} f(x)=4\text{.}\) Since \(f(2)\) is undefined, the discontinuity is removable.

    ğŸ”—

    ğŸ”—
    Solution.
    True. The left and right limits both exist but are unequal, which is the definition of a jump discontinuity.
    ğŸ”—
    ğŸ”—
    ğŸ”—
    4.
    The function \(f(x)=\sqrt{x-1}\) is continuous for all real numbers.
    ğŸ”—
    Solution.
    False. The function is only defined for \(x\geq 1\text{.}\) It is continuous on its domain \([1,\infty)\text{,}\) but not for all real numbers.
    ğŸ”—
    ğŸ”—
    ğŸ”—
    5.
    If \(f\) and \(g\) are both continuous at \(x=a\text{,}\) then \(f+g\) is also continuous at \(x=a\text{.}\)
    ğŸ”—
    Solution.
    True. Can verify using the limit laws and definition of continuity.
    ğŸ”—
    ğŸ”—
    ğŸ”—
    6.
    The piecewise function \(f(x)=\begin{cases} x+1 & x<2 \\ 3x-1 & x\geq 2 \end{cases}\) is continuous at \(x=2\text{.}\)
    ğŸ”—
    Solution.
    False. \(\displaystyle\lim_{x\to 2^-} f(x)=3\) while \(\displaystyle\lim_{x\to 2^+} f(x)=5\text{.}\) Since the one-sided limits are unequal, the function has a jump discontinuity at \(x=2\text{.}\)
    ğŸ”—
    ğŸ”—
    ğŸ”—
    7.
    If \(\displaystyle\lim_{x\to a} f(x)\) exists, then \(f\) must be continuous at \(x=a\text{.}\)
    ğŸ”—
    Solution.
    False. Continuity also requires that \(f(a)\) is defined and equals the limit. A function can have a removable discontinuity where the limit exists but the function is undefined (or defined to a different value).
    ğŸ”—
    ğŸ”—
    ğŸ”—
    8.
    The function \(f(x)=\dfrac{1}{x^2}\) has an infinite discontinuity at \(x=0\text{.}\)
    ğŸ”—
    Solution.
    True. As \(x\to 0^\pm\text{,}\) \(f(x)\to +\infty\text{.}\) The graph has a vertical asymptote at \(x=0\text{,}\) so this is an infinite discontinuity.
    ğŸ”—
    ğŸ”—
    ğŸ”—
    9.
    The domain of continuity of \(f(x)=\dfrac{\sqrt{x}}{x-1}\) is \((0,1)\cup(1,\infty)\text{.}\)
    ğŸ”—
    Solution.
    False. The square root requires \(x\ge 0\text{,}\) and the denominator excludes \(x=1\text{.}\) Thus the domain is \([0,\infty)\setminus\{1\}\text{.}\) At the endpoint \(x=0\text{,}\) we have \(f(0)=0\) and
    ğŸ”—
    \begin{equation*} \displaystyle\lim_{x\to 0^+}\frac{\sqrt{x}}{x-1}=0=f(0). \end{equation*}
    So the function is continuous at \(x=0\) (one-sided at the endpoint). Therefore the domain of continuity is \([0,1)\cup(1,\infty)\text{,}\) not \((0,1)\cup(1,\infty)\text{.}\)

    ğŸ”—

    10.
    If \(\displaystyle\lim_{x\to a^-} f(x)=\infty\) and \(\displaystyle\lim_{x\to a^+} f(x)=-\infty\text{,}\) then \(f\) has an essential discontinuity at \(x=a\text{.}\)
    ğŸ”—
    Solution.
    False. When one-sided limits are infinite (even if they diverge differently), this is classified as an infinite discontinuity, not an essential discontinuity.
