    Section 1.2 Limit Laws and Algebraic Techniques

    Objectives

    • State and apply the basic Limit Laws (sum, difference, constant multiple, product, quotient, power, root).
      🔗
    • Recognize when direct substitution can be used to evaluate a limit and when it leads to meaningless expressions.
      🔗
    • Evaluate limits that initially give meaningless expressions by algebraic manipulation
      🔗
    • Analyze limits of piecewise-defined functions by considering one-sided limits.
      🔗
    • Apply the Squeeze Theorem to evaluate limits of functions bounded above and below.
      🔗
    🔗

    Subsection 1.2.1 Operations on Limits

    Some general combination rules make most limit computations routine.
    🔗

    Theorem 1.2.1. Limit Laws.

    Suppose we know that both \(\displaystyle \lim_{x\to a} f(x)\) and \(\displaystyle \lim_{x\to a} g(x)\) exist. Then we have the following Limit Laws:
    🔗
    • Sum: \(\displaystyle \lim_{x\to a} (f(x) + g(x)) = \lim_{x\to a} f(x) + \lim_{x\to a} g(x)\text{.}\)
      🔗
    • Difference: \(\displaystyle \lim_{x\to a} (f(x) - g(x)) = \lim_{x\to a} f(x) - \lim_{x\to a} g(x)\text{.}\)
      🔗
    • Constant Multiple: \(\displaystyle \lim_{x\to a} (c f(x)) = c \cdot \lim_{x\to a} f(x)\text{,}\) for a constant \(c\text{.}\)
      🔗
    • Product: \(\displaystyle \lim_{x\to a} f(x) g(x) = (\lim_{x\to a} f(x))(\lim_{x\to a} g(x))\text{.}\)
      🔗
    • Quotient: \(\displaystyle \lim_{x\to a} \frac{f(x)}{g(x)} = \frac{\lim_{x\to a} f(x)}{\lim_{x\to a} g(x)}\text{,}\) provided \(\lim_{x\to a} g(x) \ eq 0\text{.}\)
      🔗
    • Power: \(\displaystyle \lim_{x\to a} (f(x))^n = (\lim_{x\to a} f(x))^n\text{,}\) for a whole number \(n\text{.}\)
      🔗
    • Root: \(\displaystyle \lim_{x\to a} \sqrt[n]{f(x)} = \sqrt[n]{\lim_{x\to a} f(x)}\text{,}\) for a whole number
       1 
      If \(n\) is even, we assume \(\displaystyle\lim_{x\to a} f(x) > 0\)
      \(n\text{.}\)
      🔗
    These all have the form: “The limit of an operation equals the operation applied to the limits.” These Laws are also valid for one-sided limits.
    🔗
    🔗

    Proof.

    A full proof of these laws requires high-level mathematics and the formal \(\varepsilon\)–\(\delta\) definition of limits (beyond the scope of this course). Rigorous proofs like this can be found in MSU’s MTH 320 Real Analysis course. For those who would like a glimpse a Real Analysis proof, see this video proving the Limit Sum Law
    🔗
    🔗

    Problem 1.2.2.

    Suppose \(\displaystyle\lim_{x\to 5} f(x) = 2\) and \(\displaystyle\lim_{x\to 5} g(x) = 3\text{.}\) Find \(\displaystyle\lim_{x\to 5} \left(2f(x) - (g(x))^2\right)\text{.}\)
    🔗
    Solution.
    Using the Difference, Constant Multiple, and Power Limit Laws:
    🔗
    \begin{equation*} 2\cdot \lim_{x\to 5} f(x) - \left(\lim_{x\to 5} g(x)\right)^2 = 2(2) - (3)^2 = 4 - 9 = -5. \end{equation*}
    🔗
    🔗
    🔗

    Subsection 1.2.2 Limits by Plugging In

    Assuming the Limit Laws and the Basic Limits \(\displaystyle\lim_{x\to a} x = a\) and \(\displaystyle\lim_{x\to a} c = c\text{,}\) we can prove that most functions are continuous, meaning the limit can be obtained by direct substitution.
    🔗

    Example 1.2.3. A Limit by Substitution.

    Compute \(\displaystyle \lim_{x\to 2} \frac{1-\sqrt{x}}{1+x}\) using the Limit Laws.
    🔗
    Solution.
    Step by Step:
    🔗
    \begin{equation*} \begin{array}{rcll} \displaystyle\lim_{x\to 2} \frac{1-\sqrt{x}}{1+x} \amp=\amp \displaystyle\frac{\displaystyle\lim_{x\to 2}\,1{-}\sqrt{x}}{\displaystyle\lim_{x\to 2}\,1+x} \amp \text{ by the Quotient Law } \\[2em] \amp=\amp \displaystyle\frac{\displaystyle\lim_{x\to 2}\,1-\displaystyle\lim_{x\to 2}\sqrt{x}}{\displaystyle\lim_{x\to 2}1\,+\,\displaystyle\lim_{x\to 2}x} \amp \text{ by the Sum and Difference Laws}\!\!\! \\[2em] \amp=\amp \displaystyle\frac{\displaystyle\lim_{x\to 2}\,1-\sqrt{\displaystyle\lim_{x\to 2}x}}{\displaystyle\lim_{x\to 2}1\,+\,\displaystyle\lim_{x\to 2}x} \amp \text{ by the Root Law} \\[2em] \amp=\amp \displaystyle\frac{1{-}\sqrt{2}}{1\,{+}\,2} \ =\ \frac{1{-}\sqrt{2}}{3} \amp \text{ by the Basic Limits.} \end{array} \end{equation*}
    Thus the correct limit is obtained simply by substituting \(x=2\text{.}\)
    🔗
    Note: The Quotient Law requires that the denominator have a non-zero limit. We tentatively proceed with the computation and find the denominator to be 3, which retrospectively justifies the quotient step.
    🔗
    🔗
    🔗

    Remark 1.2.4.

    Substituting \(x=a\) usually gives the correct limit unless it leads to a meaningless expression such as \(\tfrac{0}{0}\) or \(\sqrt{-1}\) (complex values are not considered in this course).
    🔗
    🔗
    In Section 2.3 and Section 2.4, we will verify that trigonometric functions like \(\sin(x)\) and \(\tan(x)\) are also continuous when defined, and the same holds for functions like \(2^x\) and \(\log(x)\text{.}\) Thus, this principle works for pretty much all reasonable functions.
    🔗
    🔗

    Subsection 1.2.3 Limits by Canceling Zeros

    Some of the most important limits are those where substitution gives a meaningless expression like \(\tfrac{0}{0}\text{.}\) To compute these, we must cancel vanishing factors until the expression can be evaluated. This often requires algebraic manipulation such as factoring or multiplying by a conjugate.
    🔗

    Example 1.2.5. Canceling Factors.

    Evaluate \(\displaystyle \lim_{x\to 2} \frac{x^2-4x+4}{x^2-x-2}\text{.}\)
    🔗
    Solution.
    Direct substitution gives the meaningless expression \(\tfrac{0}{0}\text{,}\) so we must simplify the expression first. Factor numerator and denominator:
    🔗
    \begin{equation*} \begin{array}{rcl} \displaystyle\lim_{x\to 2}\frac{(x-2)^2}{(x-2)(x+1)} \amp=\amp \displaystyle\lim_{x\to 2} \frac{x-2}{x+1} \\ \amp=\amp \frac{0}{3} = 0. \end{array} \end{equation*}
    Remark 1.2.6.
    Please note that this example is consistent with what we saw in the previous section. Here is a Desmos graph of the function, along with a table of values near \(x=2\text{.}\)
    🔗
    Keep in mind, however, that on quizzes and exams you may not have access to a calculator — in those cases the algebraic method will be our primary tool for solving such problems.
    🔗
    🔗
    🔗
    🔗

    Example 1.2.8. Using a Conjugate.

    Evaluate \(\displaystyle \lim_{x\to 9} \frac{x-9}{\sqrt{x}-3}\text{.}\)
    🔗
    Solution.
    Direct substitution gives the meaningless expression \(\tfrac{0}{0}\text{,}\) so we must simplify the expression first.
    🔗
    Multiply numerator and denominator by the conjugate \(\sqrt{x}+3\text{:}\)
    🔗
    \begin{equation*} \displaystyle\lim_{x\to 9}\frac{x-9}{\sqrt{x}-3}\cdot\frac{\sqrt{x}+3}{\sqrt{x}+3} = \lim_{x\to 9}\frac{(x-9)(\sqrt{x}+3)}{(\sqrt{x})^2-3^2} \end{equation*}
    Simplify the denominator using the difference of squares:
    🔗
    \begin{equation*} \displaystyle\lim_{x\to 9}\frac{(x-9)(\sqrt{x}+3)}{x-9} \end{equation*}
    Now cancel the common factor \(x-9\text{:}\)
    🔗
    \begin{equation*} \displaystyle\lim_{x\to 9} (\sqrt{x}+3) = 6. \end{equation*}
    Remark 1.2.9.
    Again, this algebraic approach is important, but we can also explore the behavior graphically or with tables. View it here: Desmos Graph.

    Subsection 1.2.5 Limits by Squeezing

    Some limits \(\displaystyle\lim_{x\to a} f(x)\) are difficult to evaluate because the function \(f(x)\) behaves erratically near \(x=a\text{.}\) For example:
    🔗
    \begin{equation*} \lim_{x\to 1} \Bigl(x + 4(x-1)^2 \sin\!\bigl(\tfrac{1}{x-1}\bigr)\Bigr). \end{equation*}
    The graph shows the unusual behavior, oscillating faster and faster near \(x=1\) because \(\sin(\theta)\) completes infinitely many periods as \(\theta = \tfrac{1}{x-1}\) becomes larger.
    🔗
    However, we can still find the limit using the Squeeze Theorem, which applies when \(f(x)\) is squeezed between two other functions.
    🔗

    Theorem 1.2.12. Squeeze Theorem.

    Suppose \(g(x) \leq f(x) \leq h(x)\) for all \(x\) near \(a\) (except possibly \(x=a\)), and \(\displaystyle\lim_{x\to a} g(x) = \displaystyle\lim_{x\to a} h(x) = L\text{.}\) Then \(\displaystyle\lim_{x\to a} f(x) = L\text{.}\)
    🔗
    🔗
    Now let’s apply the Squeeze Theorem to our example
    🔗

    Example 1.2.13. Squeezing Example.

    Evaluate \(\displaystyle \lim_{x\to 1} x + 4(x-1)^2\sin(\tfrac{1}{x-1})\text{.}\)
    🔗
    Solution.
    First note the expression is defined for \(x \neq 1\text{,}\) which is fine for a limit: we only need values of \(x\) near (but not equal to) \(1\text{.}\)
    🔗
    Start from the basic bound true for every real \(\theta\text{:}\)
    🔗
    \begin{equation*} -1 \;\le\; \sin(\theta) \;\le\; 1. \end{equation*}
    Substitute \(\theta = \tfrac{1}{x-1}\) (valid when \(x \neq 1\)):
    🔗
    \begin{equation*} -1 \;\le\; \sin\!\Bigl(\tfrac{1}{x-1}\Bigr) \;\le\; 1. \end{equation*}
    Multiply the entire inequality by \(4(x-1)^2\text{.}\) Because \(4(x-1)^2 \ge 0\) for all \(x\text{,}\) the direction of the inequalities is preserved (multiplying by a nonnegative number does not flip signs):
    🔗
    \begin{equation*} -\,4(x-1)^2 \;\le\; 4(x-1)^2 \sin\!\Bigl(\tfrac{1}{x-1}\Bigr) \;\le\; 4(x-1)^2. \end{equation*}
    Now add \(x\) to each part of the inequality. Adding the same quantity to all three parts preserves the inequalities:
    🔗
    \begin{equation*} x - 4(x-1)^2 \;\le\; x + 4(x-1)^2 \sin\!\Bigl(\tfrac{1}{x-1}\Bigr) \;\le\; x + 4(x-1)^2. \end{equation*}
    Define the “floor” and “ceiling” functions \(g(x)=x-4(x-1)^2\) and \(h(x)=x+4(x-1)^2\text{.}\) Compute their limits using substitution:
    🔗
    \begin{equation*} \lim_{x\to 1} g(x) \;=\; \lim_{x\to 1}\!\Bigl(x - 4(x-1)^2\Bigr) \;=\; 1 \end{equation*}
    \begin{equation*} \lim_{x\to 1} h(x) \;=\; \lim_{x\to 1}\!\Bigl(x + 4(x-1)^2\Bigr) \;=\; 1 \end{equation*}
    Since for all \(x\) near (but not equal to) \(1\) we have \(g(x) \le x + 4(x-1)^2 \sin\!\bigl(\tfrac{1}{x-1}\bigr) \le h(x)\text{,}\) and both bounding limits equal \(1\text{,}\) the Squeeze Theorem gives:
    🔗
    \begin{equation*} \lim_{x\to 1} \Bigl(x + 4(x-1)^2 \sin\!\bigl(\tfrac{1}{x-1}\bigr)\Bigr) \;=\; 1. \end{equation*}
    🔗
    🔗

    Remark 1.2.14.

    Trigonometric functions are especially good candidates for the Squeeze Theorem, since they have natural bounds. For example,
    🔗
    \begin{equation*} -1 \leq \sin(\theta) \leq 1 \end{equation*}
    and
    🔗
    \begin{equation*} -1 \leq \cos(\theta) \leq 1. \end{equation*}
    These inequalities allow us to evaluate many limits involving trigonometric functions by “squeezing” them between simpler functions.

    Exercises 1.2.6 Practice / Study Problems

    Some True/False Problems.

    Determine whether each statement is true or false and explain your reasoning
    🔗
    1.
    If \(\displaystyle\lim_{x\to 2} f(x) = 5\) and \(\displaystyle\lim_{x\to 2} g(x) = 7\text{,}\) then \(\displaystyle\lim_{x\to 2}(f(x)+g(x))=12\text{.}\)
    🔗
    Solution.
    True. By the Sum Law: \(\displaystyle\lim_{x\to 2}(f(x)+g(x))=\displaystyle\lim_{x\to 2} f(x)+\displaystyle\lim_{x\to 2} g(x)=5+7=12\text{.}\)
    🔗
    🔗
    🔗
    2.
    If \(\displaystyle\lim_{x\to 3} f(x) = 5\text{,}\) then \(f(3) = 5\text{.}\)
    🔗
    Solution.
    False. A limit concerns nearby values, not necessarily the value at the point. For example, consider Example 1.2.5. The limit exists even though the function itself is not defined at the point.
    🔗
    🔗
    🔗
    3.
    \(\displaystyle\lim_{x\to 1}\dfrac{x^2-1}{x-1} = 2\text{.}\)
    🔗
    Solution.
    True. Factor and cancel: \(\dfrac{(x-1)(x+1)}{x-1}=x+1\) for \(x\neq 1\text{,}\) so \(\displaystyle\lim_{x\to 1}(x+1)=2\text{.}\)
    🔗
    🔗
    🔗
    4.
    The Limit Laws can always be applied, even if one of the limits does not exist.
    🔗
    Solution.
    False. Each Limit Law requires the relevant limits to exist. If one does not exist, the law cannot be applied.
    🔗
    🔗
    🔗
    5.
    If \(\displaystyle\lim_{x\to 2} f(x)=0\) and \(\displaystyle\lim_{x\to 2} g(x)=0\text{,}\) then \(\displaystyle\lim_{x\to 2}\dfrac{f(x)}{g(x)}\) must equal 1.
    🔗
    Solution.
    False. The form \(\tfrac{0}{0}\) is meaningless. For example, \(f(x)=x-2\text{,}\) \(g(x)=2(x-2)\) give \(\displaystyle\lim_{x\to 2}\dfrac{f(x)}{g(x)}=\tfrac{1}{2}\text{.}\)
    🔗
    🔗
    🔗
    6.
    If \(f(x)\) is a polynomial, then \(\displaystyle\lim_{x\to a} f(x) = f(a)\) for every real number \(a\text{.}\)
    🔗
    Solution.
    True. Polynomials are continuous. We can apply the limit laws to substitute as discussed previously in this section. So \(\displaystyle\lim_{x\to a} f(x)=f(a)\) for all \(a\text{.}\)
    🔗
    🔗
    🔗
    7.
    \(\displaystyle\lim_{x\to 0}\dfrac{|x|}{x}\) does not exist.
    🔗
    Solution.
    True. \(\displaystyle\lim_{x\to 0^-}\dfrac{|x|}{x}=-1\) and \(\displaystyle\lim_{x\to 0^+}\dfrac{|x|}{x}=1\text{,}\) so the two-sided limit does not exist.
    🔗
    🔗
    🔗
    8.
    \(\displaystyle\lim_{x\to 1}\dfrac{x^2-4}{x-2} = -3\text{.}\)
    🔗
    Solution.
    False. Simplify to \(x+2\text{,}\) so \(\displaystyle\lim_{x\to 1}(x+2)=3\text{,}\) not \(-3\text{.}\)
    🔗
    🔗
    🔗
    9.
    \(\displaystyle\lim_{x\to 0} x\sin(\tfrac{1}{x}) = 0\text{.}\)
    🔗
    Solution.
    True. By the Squeeze Theorem: \(-|x|\leq x\sin(\tfrac{1}{x})\leq |x|\text{,}\) and both bounds go to 0 as \(x\to 0\text{.}\)
    🔗
    🔗
    🔗
    10.
    The Squeeze Theorem can be used to evaluate \(\displaystyle\lim_{x\to 0} x^2\cos(\tfrac{1}{x})\text{.}\)
    🔗
    Solution.
    True. Since \(-1\leq \cos(\tfrac{1}{x})\leq 1\text{,}\) multiplying by \(x^2\ge 0\) gives \(-x^2 \leq x^2\cos(\tfrac{1}{x}) \leq x^2\text{.}\) Both bounds go to 0, so \(\displaystyle\lim_{x\to 0} x^2\cos(\tfrac{1}{x})=0\text{.}\)
